= {DataPieChartName} を使用した作業の開始

== はじめに

link:{DataPieChartLink}.{DataPieChartName}.html[{DataPieChartName}] コントロールを使用して作業を開始するには、link:data-pie-chart-data-binding.html[データ バインディング] トピックを参照してください。

== 基本的な {DataPieChartName} プロパティ

以下は、円チャート コントロールの機能の基本的なプロパティの要約です。

ifdef::wpf[] 

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}itemssource.html[ItemsSource] - 円チャートのデータへの参照を格納します。

endif::wpf[]

ifdef::win-forms[]

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}datasource.html[DataSource] - 円チャートのデータへの参照を格納します。

endif::win-forms[]

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}includedproperties.html[IncludedProperties] - 値とラベルとして使用される、基になるデータ モデルのプロパティ名を指定する文字列配列。これは、以下にリストされている `ValueMemberPath` の代わりに使用できます。

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}excludedproperties.html[ExcludedProperties] - 無視される基になるデータ モデルのプロパティ名を指定する文字列配列。

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}valuememberpath.html[ValueMemberPath] - 値、つまり各円スライスの円弧の長さとして使用されるプロパティ名をデータ モデルで指定する文字列。

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}labelmemberpath.html[LabelMemberPath] - この文字列は、ラベル、つまり各円スライスに表示したり凡例内に表示するテキストとして使用されるプロパティ名をデータ モデルで指定します。

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}legend.html[Legend] - チャートに関連付けられた ItemLegend オブジェクトへのオプションの参照を格納します。

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}legendslicelabelcontentmode.html[LegendSliceLabelContentMode] - チャートにプロットされたスライスの凡例のラベルの内容を決定する列挙型。

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlightingbehavior.html[HighlightingBehavior] - チャート内のスライスまたはシリーズをどのように強調表示するかを決定する列挙型。

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlightingmode.html[HighlightingMode] - チャート内の強調表示されたスライスの外観を決定する列挙型。

* pick:[wpf="link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlighteditemssource.html[HighlightedItemsSource]"] pick:[win-forms="link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlighteddatasource.html[HighlightedDataSource]"] - チャートにプロットされたデータのサブセットを表すコレクション。データのサブセットを視覚化するための強調表示オーバーレイに使用されます。

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlightedvaluesdisplaymode.html[HighlightedValuesDisplayMode] - `HighlightedDataSource` を表すオーバーレイを有効にできる列挙プロパティ。

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}radiusextent.html[RadiusExtent] - 円チャートの半径を決定します。1 に設定すると、半径はコントロールの Width と Height の最大値に等しくなります。有効な値の範囲は 0～1 です。

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}selectionbehavior.html[SelectionBehavior] - チャートで単一選択、複数選択、または選択なしが有効かどうかを決定する列挙型。

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}selectionbehavior.html[SelectionMode] - スライスが選択されたときの動作を決定する列挙型。

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}selectedseriesitems.html[SelectedSeriesItems] - 現在選択されている円チャートのスライスを示す、各スライスの基になるデータ項目のコレクション。

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}slicelabelcontentmode.html[SliceLabelContentMode] - 各スライスのラベルに何を表示するかを指定する列挙型。

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}slicelabelpositionmode.html[SliceLabelPositionMode] - ラベルの配置を指定する列挙型。

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}startangle.html[StartAngle] - 円スライスが描画を開始するオフセット角度 (度)。

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}sweepdirection.html[SweepDirection] - StartAngle から開始する、円スライスが描画される角度方向を決定します。

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}brushes.html[Brushes] - 円スライスに色を付けるために起こった順番に使用されるブラシのコレクションへのオプション参照。
