== 次元の構成

=== 概要

link:{SurfaceChartLink}.xamscattersurface3d_members.html[XamScatterSurface3D] link:{SurfaceChartLink}.xamchart3d~aspect.html[Aspect] プロパティを使用して _xamScatterSurface3D_ コントロールの次元を構成します。

=== プロパティ設定

以下の表は、任意の構成とそれを管理するプロパティ設定のマッピングを示します。

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|設定の選択肢:

|_xamScatterSurface3D_ コントロールの次元を構成
| link:{SurfaceChartLink}.xamchart3d~aspect.html[Aspect]
|`Vector3D`

|====

=== 例

以下のスクリーンショットは、以下のコードの結果として _xamScatterSurface3D_ がどのような外観になるかを示します。

image::images/Aspect_and_Perspective_1.png[]

以下のコードはこの例を実装します。

*XAML の場合:*

[source,xaml]
----
<ig:XamScatterSurface3D Name="SurfaceChart" 
 ItemsSource="{Binding Path=DataCollection}" 
 XMemberPath="X" YMemberPath="Y" ZMemberPath="Z" 
 Aspect="1 1 0.5" />
----

*C# の場合:*

[source,csharp]
----
SurfaceChart.Aspect = new System.Windows.Media.Media3D.Vector3D(1, 1, 0.5);
----

*Visual Basic の場合:*

[source,vb]
----
SurfaceChart.Aspect = New System.Windows.Media.Media3D.Vector3D(1, 1, 0.5)
----

== 視点の構成

=== 概要

link:{SurfaceChartLink}.xamscattersurface3d_members.html[XamScatterSurface3D] link:{SurfaceChartLink}.xamchart3d~perspective.html[Perspective] プロパティを使用して _xamScatterSurface3D_ コントロールの視点を構成します。

デフォルトで、`Perspective` プロパティの初期値は 0.5 ミリ秒です。

=== プロパティ設定

以下の表は、任意の構成とそれを管理するプロパティ設定のマッピングを示します。

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|設定の選択肢:

|_xamScatterSurface3D_ コントロールの視点を構成
| link:{SurfaceChartLink}.xamchart3d~perspective.html[Perspective]
|`double`

|====

=== 例

以下のスクリーンショットは、以下のコードの結果として _xamScatterSurface3D_ がどのような外観になるかを示します。

image::images/Aspect_and_Perspective_2.png[]

以下のコードはこの例を実装します。

*XAML の場合:*

[source,xaml]
----
<ig:XamScatterSurface3D Name="SurfaceChart" ItemsSource="{Binding Path=DataCollection}" XMemberPath="X" YMemberPath="Y" ZMemberPath="Z" Perspective="1"/>
----

*C# の場合:*

[source,csharp]
----
SurfaceChart.Perspective = 1;
----

*Visual Basic の場合:*

[source,vb]
----
SurfaceChart.Perspective = 1
----
