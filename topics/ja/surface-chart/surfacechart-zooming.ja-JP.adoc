== チャート スケールの構成

=== 概要

link:{SurfaceChartLink}.xamscattersurface3d_members.html[XamScatterSurface3D] link:{SurfaceChartLink}.xamchart3d~scale.html[Scale] プロパティを使用して _xamScatterSurface3D_ コントロールのスケールを構成します。

デフォルトで、`Scale` プロパティの初期値は 0.8 ミリ秒です。

=== プロパティ設定

以下の表は、任意の構成とそれを管理するプロパティ設定のマッピングを示します。

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|設定の選択肢:

|_xamScatterSurface3D_ スケールを構成します。
| link:{SurfaceChartLink}.xamchart3d~scale.html[Scale]
|`double`

|====

=== 例

以下の設定は、以下のコードの結果として _xamScatterSurface3D_ コントロールの外観がどのようになるかを示しています。

[options="header", cols="a,a"]
|====
|プロパティ|値

| link:{SurfaceChartLink}.xamchart3d~scale.html[Scale]
| _0.5_ 

|====

image::images/Zooming_1.png[]

以下のコードはこの例を実装します。

*XAML の場合:*

[source,xaml]
----
<ig:XamScatterSurface3D Name="SurfaceChart3" 
 ItemsSource="{Binding Path=DataCollection}" 
 XMemberPath="X" YMemberPath="Y" ZMemberPath="Z" 
 Scale="0.5"/>
----

*C# の場合:*

[source,csharp]
----
…
SurfaceChart.Scale = 0.5;
----

*Visual Basic の場合:*

[source,vb]
----
…
SurfaceChart.Scale = 0.5
----
