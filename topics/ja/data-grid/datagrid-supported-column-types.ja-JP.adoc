= サポートされる列タイプ

このトピックでは、link:{DataGridLink}.{DataGridName}.html[{DataGridName}] コントロールのサポートされる列タイプを紹介します。

列の作業に関する詳細な説明は、「 link:datagrid-working-with-columns.html[列の作業]」のトピックを参照してください。

==== サポートされる列タイプ:

* link:{DataGridLink}.textcolumn_members.html[TextColumn] は、関連セルのフォーマット済みテキストの表示に使用されます。これは、文字列データ型のデータを表示する場合に使用されるデフォルトの列タイプです。
* link:{DataGridLink}.numericcolumn_members.html[NumericColumn] は、フォーマット済み数値の表示に使用され、セル内で小数位置の配置を制御します。最小および最大のしきい値の設定や小数桁数の表示を可能にします。
* link:{DataGridLink}.datetimecolumn_members.html[DateTimeColumn] は、セル内に pick:[android="`Calendar`"]pick:[xaml="`DateTime`"] オブジェクトを表示するために使用します。`DateTimeLong`、`DateTimeShort`、`DateLong`、`DateShort`、`TimeLong`、および `TimeShort` を含む事前定義済みの DateTime フォーマットを提供します。
* link:{DataGridLink}.imagecolumn_members.html[ImageColumn] は、セル内に画像を表示し、セルでの画像引き伸ばしをカスタマイズできるオプションを公開します。
* link:{DataGridLink}.templatecolumn_members.html[TemplateColumn] は、セル テンプレートとともに使用すると、列のセル配置にカスタム テンプレートを適用できます。
