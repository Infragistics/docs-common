= API の概要

このセクションは、 link:{DataGridLink}.{DataGridName}.html[{DataGridName}] コントロールに関連する各名前空間をリストします。また、このコントロールでコードを書く場合に使用するいくつかのキー クラスもあります。このページの名前空間とクラスはインフラジスティックス API マニュアルに直接リンクします。
