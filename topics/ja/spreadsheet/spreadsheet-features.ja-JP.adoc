== 概要

以下のスクリーンショットは、複数のスプレッドシート データを表示する  _{SpreadsheetName}_   コントロールを示します。

=== {SpreadsheetName} の概要

{SpreadsheetName} コントロールでは、 pick:[xaml="link:excelengine-about-infragistics-excel-engine.html[Infragistics Excel Engine]"]  pick:[win-forms="link:waw-infragistics-excel-engine.html[Infragistics Excel Engine]"]  でサポートされるデータ モデルによって表されるスプレッドシート データの表示と編集ができます。データ モデルには、 link:{ApiPlatform}documents.excel{ApiVersion}~infragistics.documents.excel.workbook.html[ワークブック]、 link:{ApiPlatform}documents.excel{ApiVersion}~infragistics.documents.excel.worksheet.html[ワークシート]、 link:{ApiPlatform}documents.excel{ApiVersion}~infragistics.documents.excel.worksheetcell.html[セル]、 link:{ApiPlatform}documents.excel{ApiVersion}~infragistics.documents.excel.formula.html[数式]などが含まれます。

以下のスクリーンショットは、複数のスプレッドシート データを表示する {SpreadsheetName} コントロールを示します。

[[_Ref395877610]]

=== セルの配置

コントロールは多数の水平および垂直のセルのコンテンツの配置をサポートします。段落、テキストの折り返し、セルに合わせた縮小、セルのマージもサポートされます。

注:

[NOTE]
====

ifdef::xaml[]

回転はサポートされていません。
endif::xaml[]

ifdef::win-forms[]

回転および左右に整列はサポートされていません。
endif::win-forms[]

====

[[_Ref395876419]]

=== セル境界線

コントロールは、異なったスタイルと色を持つセルの境界線および端をサポートします。斜線もサポートします。ズーム操作中、コントロールは境界線を拡大縮小しません。

[[_Ref395874256]]

=== セルの塗りつぶし

コントロールは、塗りつぶし、グラデーションの塗りつぶし、パターン色、パターン スタイルなど、さまざまなセルの塗りつぶしをサポートします。

[[_Ref396145620]]

=== セルのインプレース編集

コントロールは、編集中のセルの上に配置されたエディターを使用する多機能の「インプレース」セル編集をサポートします。ユーザーは、セルの書式付き文字列または数式を編集できます。

==== 関連トピック:

link:spreadsheet-uiu-cell-editing.html[セル編集のインタラクション ({SpreadsheetName})]

ifdef::xaml[]

[[_Ref395886158]]

=== コマンド

コントロールは、さまざまな機能を起動するためのコマンドを提供します。

==== 関連トピック:

link:spreadsheet-work-commands.html[コマンドの操作 ({SpreadsheetName})]

[[_Ref395881898]]

=== コンテキスト メニュー

コントロールは、コンテキスト メニューを提供します。これにより、ユーザーは、選択された視覚要素に応じて、異なった操作を行うことができます。

==== 関連トピック:

link:spreadsheet-uiu-context-menu.html[コンテキスト メニューのインタラクション ({SpreadsheetName})]

endif::xaml[]

[[_Ref395881763]]

=== コピー / 貼り付け

コントロールは、セル間、シート間、さらに外部のアプリケーション間 (MS Excel など) の情報のクリップボードによるコピーをサポートします。

==== 関連トピック:

link:spreadsheet-work-clipboard.html[クリップボードの操作 ({SpreadsheetName})]

[[_Ref395880031]]

=== データ検証

コントロールは、セルの検証ルール セットに基づいてユーザー入力の検証を可能にするデータ検証機能をサポートします。ルールの構成は、ユーザー入力が検証をパスしなかった場合に表示する入力メッセージとエラー メッセージ ダイアログも含みます。

==== 関連トピック:

link:spreadsheet-uiu-data-validation.html[データ検証のインタラクション ({SpreadsheetName})]

[[_Ref395880032]]

=== フォント スタイル

コントロールがサポートするテキスト プロパティは、フォント ファミリ、フォント サイズ、太字、斜体、下線、二重下線、取り消し線およびカラーです。コントロールは、フォントの高さではなく、行全体の高さに基づいて、上付き文字と下付き文字を表示します。

注:

[NOTE]
====
下線、小さい上付き、または下付きテキストと大きいテキストが同じ行にある場合の、MS Excel との表示の違いを確認してください。
====

[[_Ref396117365]]

=== 数式バー

コントロールにより、ユーザーはセル テキストと数式を編集できます。数式バーの名前ボックスを使用して、ユーザーは名前付き参照を定義し、適切なセルまたはセルの範囲に素早く移動できます。数式バーは、複数行のコンテンツの編集もサポートします。

==== 関連トピック:

link:spreadsheet-uiu-formula-bar.html[数式バーのインタラクション ({SpreadsheetName})]

[[_Ref395885169]]

=== ペインのフリーズ

コントロールは、先頭行 / 左の列のフリーズを可能にします。スクロール中、フリーズされた行 / 列は表示されます。

==== 関連トピック:

pick:[xaml="link:excelengine-freeze-and-split-panes.html[ペインのフリーズおよび分割 (Infragistics Excel Engine)]"]
pick:[win-forms="link:excelengine-freeze-rows-and-columns.html[行と列をフリーズ (Infragistics Excel Engine)]"]

[[_Ref395865751]]

=== グリッド線

コントロールは、ワークシートのセルの分離に使用されるグリッド線を表示　/　非表示にできます。点線は、カスタム カラーがグリッド線の色として使用されている場合に描かれます。境界線または背景が存在する場合、グリッド線は表示されません。

==== 関連トピック:
pick:[xaml="link:excelengine-configuringgridlines.html[グリッド線の構成 (Infragistics Excel Engine)]"]
pick:[win-forms="link:excelengine-applying-styles-to-cells.html[スタイルをセルに適用]"] 

ifdef::xaml[]

[[_Ref395865766]]

=== ヘッダー

コントロールは、列ヘッダーと行ヘッダーを表示　/　非表示にできます。

==== 関連トピック:

link:excelengine-configuringheaders.html[ヘッダーの構成 (Infragistics Excel Engine)]

[[_Ref395887540]]

=== 非表示

コントロールは列と行の非表示をサポートします。ユーザーは列または行のサイズ変更を開始し、列　/　行が表示されなくなるまで縮小することができます。非表示になった列または行がある位置に、特別なインジケータが表示されるため、列　/　行を再び表示することができます。

==== 関連トピック:

link:spreadsheet-uiu-columns-and-rows.html[行と列のインタラクション ({SpreadsheetName})]

endif::xaml[]

[[_Ref395887542]]

=== ハイパーリンク

コントロールは、基本オブジェクト モデルで定義されるハイパーリンク、およびワークシートのセル数式で定義されるハイパーリンクをサポートします。

注:

[NOTE]
====
ハイパーリンクは図形でサポートされていません。
====

==== 関連トピック:

ifdef::win-forms[]
link:excelengine-adding-a-hyperlink-to-a-cell-in-an-excel-file.html[Excel ファイルのセルにハイパーリンクを追加]
endif::win-forms[]

ifdef::xaml[]
* link:spreadsheet-uiu-hyperlinks.html[ハイパーリンクのインタラクション ({SpreadsheetName})]
* link:spreadsheet-work-hyperlinks.html[ハイパーリンクの操作 ({SpreadsheetName})]

[[_Ref395887541]]

=== セル、列、行の挿入と削除

コントロールは、プログラムによってセル、列、行の管理をサポートします。

==== 関連トピック:

link:excelengine-inserting-and-deleting-cells.html[セル、列、行の挿入と削除 (Infragistics Excel Engine)]

endif::xaml[]

[[_Ref395887547]]

=== サイズ変更

コントロールは列と行のサイズ変更をサポートします。ユーザーは 2 つの列ヘッダーまたは 2 つの行ヘッダー の間の境界線にマウスをドラッグして、列と行のサイズを変更できます。また、自動サイズ変更は列ヘッダーの右の境界線をダブルクリック、または行ヘッダーの下の境界線をダブルクリックすることにより、サポートされます。

==== 関連トピック:

link:spreadsheet-uiu-columns-and-rows.html[行と列のインタラクション ({SpreadsheetName})]

[[_Ref395886043]]

=== 図形のサポート

コントロールはさまざまな種類のオブジェクト (図形、画像) をサポートします。オブジェクトは、独立した (セルのコンテンツから) 層で描画され、それらが置かれているセルに基づいて、移動　/　サイズ変更を行うことができます。オブジェクトの回転および異なる変換 (左右反転または上下反転など) はサポートされます。

注:

[NOTE]
====
現在、ユーザー インターフェイスを使用して図形を操作することはできません。
====

==== 関連トピック:

*  pick:[xaml="link:excelengine-adding-a-shape-to-an-excel-worksheet.html[図形をワークシートに追加 (Infragistics Excel Engine)]"]  pick:[win-forms="link:excelengine-adding-shapes-to-a-worksheet.html[形状をワークシートに追加 (Infragistics Excel Engine)]"] 
*  pick:[xaml="link:excelengine-addingimagetoworksheet.html[画像をワークシートに追加 (Infragistics Excel Engine)]"]  pick:[win-forms="link:excelengine-add-an-image-to-a-worksheet.html[画像をワークシートに追加 (Infragistics Excel Engine)]"] 

[[_Ref395797889]]

=== 選択

コントロールは、プログラムまたはユーザーにより切り替えが可能な複数の選択モードをサポートします。現在選択されているセルは、プログラムによって設定および取得できます。

==== 関連トピック:

* link:spreadsheet-uiu-selection.html[選択のインタラクション ({SpreadsheetName})]
* link:spreadsheet-conf-selection.html[選択の構成 ({SpreadsheetName})]
* link:spreadsheet-work-selection.html[選択の操作 ({SpreadsheetName})]

ifdef::xaml[]

[[_Ref395884699]]

=== ペインの分割

コントロールは、ワークシート ビューを複数のペインに分割することをサポートします。スクロール可能なペインの行と列には固有のスクロールバーがあります。

==== 関連トピック:

link:excelengine-freeze-and-split-panes.html[ペインのフリーズおよび分割 (Infragistics Excel Engine)]

endif::xaml[]

[[_Ref396146031]]

=== タブ バー領域

コントロールは、ワークシートを切り替えるための専用のタブ領域をサポートします。

==== 関連トピック:

* link:spreadsheet-uiu-tab-bar-area.html[タブ バー領域のインタラクション ({SpreadsheetName})]
* link:spreadsheet-conf-tab-bar-area.html[タブ バー領域の構成 ({SpreadsheetName})]

[[_Ref395883789]]

=== 元に戻す / やり直し

コントロールは、IG Undo/Redo Framework を使用して、無制限な元に戻す / やり直し操作を提供します。元に戻す /やり直しは有効 / 無効にできます。また、各  _{SpreadsheetName}_   コントロールに対して個別または共有のやり直しマネージャーを使用することもできます。

==== 関連トピック:

link:spreadsheet-work-undo-redo.html[元に戻す / やり直しの操作 ({SpreadsheetName})]

[[_Ref395885466]]

=== ワークシートの背景

コントロールは、ワークシートの背景画像の表示をサポートします。画像は、ワークシートの背景にタイル表示されます。ズーム機能は、背景画像に影響を与えます。ペインがスクロールすると、背景画像もスクロールします。

ifdef::xaml[]

[[_Ref395883004]]

=== ズーム

コントロールは、コンテンツのズームをサポートします。ワークブックのコンテンツすべて (テキスト、画像、図形、図形の境界線) が拡大縮小されます。

注:

[NOTE]
====
セルの境界線とセルのグリッド線は拡大縮小されません。
====

==== 関連トピック:

link:excelengine-zoomlevel.html[ワークシートのズーム レベルの設定 (Infragistics Excel Engine)]

endif::xaml[]
