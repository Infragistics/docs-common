== ナビゲーションの構成の概要

=== ナビゲーションの構成の概要

アクティベーションとナビゲーションに関するトピックで説明したとおり、{SpreadsheetName} コントロールは *エンド モード* と *スクロール ロック* の 2 つの追加ナビゲーション モードをサポートします。

*  *エンド モード*  - このモードでは、矢印キーを使用して、データが設定された隣接するセルをナビゲートします。
*  *スクロール ロック*  － このモードでは、矢印キーを使用して、アクティブなセルをナビゲートするのではなく、ワークシートに表示されているコンテンツをスクロールします。

|====

|スクロール ロック ナビゲーション モードを有効または無効にします。
|なし

[NOTE]
