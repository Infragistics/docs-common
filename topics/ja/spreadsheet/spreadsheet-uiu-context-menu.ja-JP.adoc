== 概要

=== コンテキスト メニューの概要

{SpreadsheetName} コントロールが提供するコンテキスト メニューは、コントロール要素を右クリックすると表示されます。コンテキスト メニューにより、ユーザーは、右クリックした要素に応じて、異なった操作ができます。

以下の  _{SpreadsheetName}_   要素には特別なコンテキスト メニューがあります。

* 数式エディター
* ワークシートの列または行
* ワークシートのセル
* 編集モードのワークシートのセル (インプレース エディター)
* ワークシートのタブ バー領域

注:

[NOTE]
====
ユーザーが複数のワークシートを選択し、アクティブなワークシートのセルでクリップボード操作を実行した場合、選択されたすべてのワークシートに影響します。切り取り操作は、選択されたすべてのワークシートからクリップボードにセルのコンテンツを移動します。コピー操作は、選択されたすべてのワークシートからセルのコンテンツを複製します。貼り付け操作は選択されたすべてのワークシートの指定されたセルにクリップボードのコンテンツをコピーします。
====

[[_Ref396316984]]
== ワークシートの列または行のコンテキスト メニュー

=== 列または行のコンテキスト メニューの概要

ワークシートの列または行のコンテキスト メニューにより以下の操作ができます。

* 選択した列または行でのクリップボード操作を実行
* 行または列またはそのコンテンツの挿入 / 削除
* 列と行の非表示 / 非表示解除および自動サイズ設定
* 選択された列または行のハイパーリンクの削除。これはセルのワークシート ハイパーリンクのみを削除し　(数式ハイパーリンクは削除されない)、セルのデータ コンテンツはそのまま残します。

== ワークシートのセルのコンテキスト メニュー

=== ワークシートのセルのコンテキスト メニューの概要

ワークシートのセルのコンテキスト メニューによりユーザーは以下のことができます。

* セルに対するクリップボード操作を実行
* 新しい空のセルを挿入
* セルを削除またはセルのコンテンツのみを削除
* ハイパーリンクをオープン (ハイパーリンクを持つ単一のセルを選択した場合にのみ使用可能)
* ハイパーリンクを削除 (少なくとも 1 つのワークシート ハイパーリンクが選択されている場合のみ)

注:

[NOTE]
====
数式ハイパーリンクは、「ハイパーリンクの削除」コンテキスト メニュー オプションで削除できません。
====

== ワークシートのタブ バー領域のコンテキスト メニュー

=== ワークシートのタブ バー領域のコンテキスト メニューの概要

ワークシートのタブ バー領域のコンテキスト メニューにより、ユーザーは以下を実行できます。

* 新しいワークシートの挿入
* 既存のワークシートの削除
* 既存のワークシートの名前の変更
* すべてのワークシートを選択
* すべてのワークシートを選択解除 (「グループ化解除」メニュー項目)
