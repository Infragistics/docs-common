= Treemap グラフの色を設定

ツリーマップ グラフのノードの色を設定することは、ツリーマップ グラフのデザインの重要な部分です。エンド ユーザーに常に変更を警告できるからです。エンド ユーザーが特に重要なデータのクラスタを識別する際にも役立ちます。

たとえば、ツリーマップ グラフが株式にマップされており、緑が株価の上昇を示し、赤が株価の下落を示す場合です。エンド ユーザーは即座にどの株式が値上がりし、どの株式が値下がりしているのかが分かります。

以下のリンクをクリックして、ツリーマップ グラフの色の設定を学習してください。

* link:chart-set-a-treemap-charts-color-based-on-hierarchical-level.html[階層レベルに基づいてツリーマップ グラフの色を設定]

どのデータ ソースを使用しているかによって、リーフ ノードの色の設定は若干異なります。以下のリンクをクリックして、異なるデータ ソースを使用してリーフ ノードの色を設定する方法を学習してください。

* link:chart-map-a-leaf-nodes-color-to-a-specific-column-in-a-data-table.html[データ テーブルでリーフ ノードの色を特定の列にマップ]
* link:chart-set-a-leaf-nodes-color-using-a-treemapseries-object.html[TreeMapSeries オブジェクトを使用してリーフ ノードの色を設定]
