= 吹き出し注釈

吹き出し注釈では、スピーチ バブルの形を使用して、位置とテキスト表示の両方を示します。

[NOTE]
====
*注* ：グラフ注釈の作成手順の例は、 link:chart-create-annotations.html[「注釈の作成」]を参照してください。
====
