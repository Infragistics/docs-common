= 散布図-折れ線グラフについて

image::images/Chart_About_Scatter_Line_Charts_01.png[]

link:chart-scatter-chart.html[散布図]は、複数の異なるデータのシリーズ内の項目間の関係を示すか、または 2 つの値セットを x/y 座標の 1 シリーズとしてプロットする場合に使用します。散布図では、データの不均等な散らばり方やデータの塊に注意が向けられます。この種類のグラフは、科学データのプロットによく用いられ、予測結果からの収集データの偏差を強調表示できます。

link:chart-line-chart-2d.html[折れ線]グラフは等間隔のデータ値の変化のトレンドを示します。

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[散布図 - 折れ線グラフ]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[散布図 - 折れ線グラフ]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[散布図 - 折れ線グラフ]"] は、両方のグラフの種類を組み合わせたもので、個々の値とトレンドの両方を示します。
