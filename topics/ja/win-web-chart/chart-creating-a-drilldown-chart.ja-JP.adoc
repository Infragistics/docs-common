= ドリルダウン チャートの作成

チャート ドリルは、そのデータを構成する詳細を子チャートに表示できるようにエンド ユーザーがチャートのデータ要素を操作することを可能にする概念です。チャート ドリルは通常、エンド ユーザーがチャートのデータ要素をクリックできるようにしてそれによってその要素を構成する詳細をフェッチし次に同じチャートに移植されるように実装されます。この記事は、Infragistics WinChart でこの機能を達成するひとつの方法を検討します。

チャートの作成プロセスをガイドする複数部から構成される詳細なガイドは以下のトピックで構成されます。これにより 3 階層をドリルダウンすることができます。
