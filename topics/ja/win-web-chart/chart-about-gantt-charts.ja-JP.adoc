= ガント チャートについて

image::images/Chart_About_Gantt_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ガント チャート]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ガント チャート]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ガント チャート]"] は、浮動バーを表示します。各棒は、定義された開始時間と終了時間を描写します。ガント チャートは、プロジェクト管理によく使用され、各棒は達成予定の個々のタスクを表します。

このアーキテクチャによって、アプリケーション開発者はチャートを簡単かつ非常に柔軟に拡張できます。Chart のレイヤ フレームワークでは、アプリケーション開発者はカスタム レイヤを既存のレイヤ セットに挿入できます。

チャートがデータを時系列で表示して、経過時間に従って x 軸で点をプロットするようにしたい場合には、 link:chart-display-data-on-a-time-scale-axis.html[「時間のスケール軸でデータを表示」]を参照してください。
