= パレート図データでの作業

このトピックでは、パレート図でデータが適切に描画されることを保証する支援となる役に立つ情報を説明します。

== データ要件

Chart コントロールによって固有のカスタム データにチャートを簡単にポイントすることができますが、そのチャートが必要とするデータの適切な量とタイプを提供することが重要です。使用しているチャートのタイプに基づいた最小要件をデータが満たさないと、エラーが生成されます。

以下は、パレート図のデータ要件のリストです。

* データ セットには、ひとつまたは複数の数値列が含まれます。
* パレート図を Series オブジェクトにバインドする場合には、Series バインディング要件の詳細は link:chart-requirements-for-series-binding.html[「Series バインディングの要件」]を参照してください。
* link:chart-composite-chart.html[合成チャート]でパレート図を使用する場合には、チャートの軸要件の詳細は、 link:chart-axis-requirements-for-composite-charts.html[「合成チャートの軸要件」]を参照してください。

[NOTE]
====
*注：* 使用できるデータが、数値列がひとつで列が多数ある書式の場合、行と列を交換する必要があります。この場合、各行の文字列の列の値が項目ラベルとして使用されます。これを実行する詳細は、 link:chart-swap-rows-and-columns.html[「行と列の入れ替え」]を参照してください。
====

== データをパレート図にマップ

チャート データは以下の規則を使用して描画されます。

* データの最初の行だけがプロットされます。
* 列名が項目ラベルとして使用されます。
