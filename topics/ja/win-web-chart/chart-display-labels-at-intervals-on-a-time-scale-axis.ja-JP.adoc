= 時間のスケール軸で間隔でラベルを表示

時間のスケール軸を数値軸として使用すると、デフォルトの  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkstyle.html[TickmarkStyle]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkstyle.html[TickmarkStyle]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkstyle.html[TickmarkStyle]"]  が Percentage に指定され、デフォルトの  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkpercentage.html[TickmarkPercentage]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkpercentage.html[TickmarkPercentage]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkpercentage.html[TickmarkPercentage]"]  が 10 に設定されます。これにより、データ範囲が 10% 刻みの目盛りで表示されます。しかし、多くの場合時間は 10 を単位としないので、これが最適結果とはなりません。その場合には、スマート目盛りを使用します。DateTime 軸の TickmarkStyle を Smart に設定すると、時間に関する適切な間隔（日数、週数、月数）で目盛りが作成されます。

スマート目盛りを有効にすると、与えられた範囲のデータに対して  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkintervaltype.html[TickmarkIntervalType]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkintervaltype.html[TickmarkIntervalType]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkintervaltype.html[TickmarkIntervalType]"]  および  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkinterval.html[TickmarkInterval]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkinterval.html[TickmarkInterval]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkinterval.html[TickmarkInterval]"]  プロパティが自動的に最適値に設定されます。完全なカスタマイズを行うには、TickmarkStyle を DataInterval に設定してこれらのプロパティを使用します。目盛りを 3 日間置きに設定するには、TickmarkIntervalType プロパティを Days に設定し、TickmarkInterval プロパティを 3 に設定します。

チャート データを時間のスケール軸に表示する方法については、 link:chart-display-data-on-a-time-scale-axis.html[「時間の連続的スケール軸へのデータの表示」]を参照してください。
