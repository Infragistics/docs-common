= 特定の注釈を表示する

注釈の表示を変更するには、 pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.annotationsappearance~visible.html[Visible]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.annotationsappearance~visible.html[Visible]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.annotationsappearance~visible.html[Visible]"]  プロパティを True に設定します。

*Visual Basic の場合：*

----
Imports Infragistics.UltraChart.Resources.Appearance
...
Dim myCallout As CalloutAnnotation = CType(Me.ultraChart1.Annotations(0), CalloutAnnotation);
myCallout.Visible = Not myCallout.Visible
----

*C# の場合：*

----
using Infragistics.UltraChart.Resources.Appearance;
...
CalloutAnnotation myCallout = (CalloutAnnotation)this.ultraChart1.Annotations[0];
myCallout.Visible = !myCallout.Visible;
----
