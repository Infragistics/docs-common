= 柱状折れ線グラフについて

image::images/Chart_About_ColumnLine_Charts_01.png[]

link:chart-column-chart-2d.html[柱状グラフ]は項目間の比較を示すか、一定期間におけるデータの変化を示します。各値は垂直方向に編成され、水平方向にカテゴリ化されます。 link:chart-line-chart-2d.html[折れ線グラフ]は等間隔でデータのトレンドを示します。

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[柱状折れ線グラフ]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[柱状折れ線グラフ]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[柱状折れ線グラフ]"] は、両方のグラフの種類を組み合わせたもので、カテゴリ化した各部分とトレンドの両方を示します。
