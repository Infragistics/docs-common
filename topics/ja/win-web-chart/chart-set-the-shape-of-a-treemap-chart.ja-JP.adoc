= Treemap グラフの形状を設定

ツリーマップ グラフのノードの図形を定義するために Chart の  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.treemapchartappearance~treemaptype.html[TreeMapType]"]  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.treemapchartappearance~treemaptype.html[TreeMapType]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.treemapchartappearance~treemaptype.html[TreeMapType]"]  プロパティを設定できます。

TreeMapType プロパティを以下に設定できます:

* Circular -- ツリーマップ グラフのノードは円形で表示されます。
* Rectangular -- ツリーマップ グラフのノードは四角形で表示されます。
* Rings -- ツリーマップ グラフのノードは輪で表示され、輪の各セクションがノードに対応します。各セクションの子ノードは次の輪の新しいセクションとして描画されます。

以下の例のノードは、ツリーマップ グラフで円形ノードを表示する方法を示します。

image::images\Chart_Set_the_Shape_of_a_Treemap_Chart.png[]

*Visual Basic の場合：*

----
Me.ultraChart1.TreeMapChart.TreeMapType = 
Infragistics.UltraChart.Shared.Styles.TreeMapType.Circular;
----

*C# の場合：*

----
this.ultraChart1.TreeMapChart.TreeMapType = 
Infragistics.UltraChart.Shared.Styles.TreeMapType.Circular;
----
