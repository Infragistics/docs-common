= 立体折れ線グラフについて

image::images/Chart_About_3D_Line_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[立体折れ線グラフ]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[立体折れ線グラフ]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[立体折れ線グラフ]"]  は、一定期間の変化の量を強調する、または複数の項目を比較します。データ ポイントは等間隔で連続してプロットされ、ポイント間の関係を強調するために線で結ばれます。

視覚的な強調をグラフに追加するために立体スタイルが使用されます。データをプロットするために余分な次元を提供しません。ただし、データ セットが似ている場合、立体折れ線グラフは視覚的な混乱を排除する支援ができます。平面折れ線グラフで別のデータ項目のシリーズに重複して分かりにくいデータ項目のシリーズは、立体折れ線グラフで表示すると簡単に見分けることができます。
