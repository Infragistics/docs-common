= 平面エリア グラフについて

image::images/Chart_About_2D_Area_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[平面エリア グラフ]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[平面エリア グラフ]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[平面エリア グラフ]"] は、一定期間の変化の量を強調する、または複数の項目を比較します。また、プロットした各値の合計を表示して全体に対する各部分の関係も示します。

エリア グラフは、折れ線グラフの 1 形式ですが、x 軸（水平軸）とデータ マーカーを結ぶAxAx間のエリアを色で塗りつぶします。これにより、複数のデータ系列で取り囲まれてデータ点が重なっている個所がはっきりわかります。

チャートがデータを時系列で表示して、経過時間に従って x 軸で点をプロットするようにしたい場合には、 link:chart-display-data-on-a-time-scale-axis.html[「時間のスケール軸でデータを表示」]を参照してください。
