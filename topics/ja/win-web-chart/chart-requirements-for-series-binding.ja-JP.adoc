= Series バインディングの要件

以下の表は、Series オブジェクトをバインドするために各チャート タイプのデータセットに含まなければならないシリーズの数とタイプの概要を述べます。これらの要件が満たされない場合、チャートは描画されません。

Chart の Series コレクションの概要は、 link:chart-series-collection.html[「Series コレクション」]を参照してください。

[NOTE]
====
*注：* 柱状折れ線グラフと散布図は link:chart-series-collection.html[Series コレクション]をサポートしません。代わりに、複数のチャート レイヤを持つ link:chart-composite-chart.html[合成チャート]を使用します。
====
