= ドリルダウン チャートの作成（1/5）

開始する前に、この詳細なガイドのために設定する規則が複数あります。

*有限のドリル レベル*

ドリル レベルは以下の 3 つです。

* レベル 1 -- Customer エンティティを表します。
* レベル 2 -- Order エンティティを表します。
* レベル 3 -- Order Detail エンティティを表します。

データは、SQL Server のインスタンスに配置されている Northwind Database から取得されます。

*ドリル パス*

レベルにドリルダウンする時には、順々にドリルすることに制限されます。ただし、上のレベルにドリルバックすると、レベルをスキップできます。

たとえば、Customers から Orders そして Order Details にドリルダウンする必要があります。上のレベルにドリルバックすると、Order Details から Customers にスキップできます。これは Order Details にドリルダウンするためには表示したい Order Details の Customer ID と Order ID を知る必要があるからです。上のレベルにドリルすると、Customer ID はすでに分かっているため、Order Details から Customers にスキップできます。
