= ログ ゼロ値をプロット

例外がスローされることなく Log(0) 値をプロットできるようにするために、軸が Log(0) をプロットしようとする時に Chart コンポーネントによって値を指定することができます。

以下のコード サンプルによって、対数 y 軸を使用して、ゼロ値を持つテーブルまたはシリーズをプロットできます。軸の最小値は 1.0 ですべてのゼロ値は 1.0 にプロットされます。

*Visual Basic の場合：*

----
Me.UltraChart1.Axis.Y.LogZero = 1.0
----

*C# の場合：*

----
this.UltraChart1.Axis.Y.LogZero = 1.0;
----

[NOTE]
====
*注:* LogZero プロパティのデフォルト値は、double.NaN です。このプロパティをデフォルト値にしておくと、例外がスローされます。
====
