= 目盛

目盛は、軸上にラベルを描画する点です。これに加えて、グリッド線の位置は目盛の位置によって制御されます。軸の各目盛の間隔を軸の  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkstyle.html[TickmarkStyle]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkstyle.html[TickmarkStyle]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkstyle.html[TickmarkStyle]"] 、 pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkinterval.html[TickmarkInterval]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkinterval.html[TickmarkInterval]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkinterval.html[TickmarkInterval]"] 、および  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkpercentage.html[TickmarkPercentage]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkpercentage.html[TickmarkPercentage]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~tickmarkpercentage.html[TickmarkPercentage]"]  プロパティを使用してカスタマイズできます。

image::images\Chart_Axes_13.png[]

[source,docX-root]
----
UltraChart1.Axis.Y.TickmarkStyle = AxisTickStyle.DataInterval
UltraChart1.Axis.Y.TickmarkInterval = 20.00
----

image::images\Chart_Axes_14.png[]

軸の長さの  _n_  パーセントごとに軸ラベルを表示するには、TickmarkStyle を Percentage に設定し、TickmarkPercentage プロパティの値を  _n_  に設定します。

データ間隔で表示される目盛の場合:

* 数値軸の場合、間隔  _n_  で軸ラベルを表示するには、TickmarkStyle を DataInterval に設定し、TickmarkInterval プロパティを  _n_  に設定します。
* SetLabelAxis の場合、 _n_  番目ごとに軸ラベルを表示するには、TickmarkStyle を DataInterval に設定し、TickmarkInterval プロパティを  _n_  に設定します。
----
日付/時間軸についての詳細は、 link:chart-display-labels-at-intervals-on-a-time-scale-axis.html[「時間のスケール軸で間隔でラベルを表示」]を参照してください。
