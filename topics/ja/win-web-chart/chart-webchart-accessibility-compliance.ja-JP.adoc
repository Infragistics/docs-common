= WebChart アクセシビリティの遵守

弊社の ASP.NET コントロールおよびコンポーネントはすべて、1973 年のリハビリテーション法第 508 条第 1194 部 22 条を遵守しています。以下は、第 1194 部 22 条の 16 の規則と各規則を遵守する方法を含んだ表です。場合によって、プロパティを設定する必要があります。それ以外の場合には、コントロールがすべての作業を実行するため、ユーザーは何もする必要がありません。

[options="header", cols="a,a"]
|====
|規則|規則を遵守する方法

|(a)
|pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.webui.ultrawebchart.ultrachart~section508.html[Section508]"] pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.webui.ultrawebchart.ultrachart~section508.html[Section508]"] プロパティを有効にすることによって、アクセシビリティを遵守するために、第 508 条に関連する外観条項を設定できます。

*注：* Section508 を有効にしたときのチャートでのツールチップの動作をよく見てください。 pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.section508chartimage~altformatstring.html[AltFormatString]"] pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.section508chartimage~altformatstring.html[AltFormatString]"] プロパティが使用されると、オリジナルのチャートのツールチップを分かりにくくする 2 番目のツールチップが生成されます。アクセシビリティを最大限にするために AltFormatString を使用してすべてのツールチップ情報を表示し、Tooltips.Display プロパティを Never に設定することにより、標準のチャートのツールチップをオフにするのが最善です。
