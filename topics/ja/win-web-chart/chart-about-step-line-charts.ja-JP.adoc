= ステップ折れ線グラフについて

image::images/Chart_About_Step_Line_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ステップ折れ線グラフ]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ステップ折れ線グラフ]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ステップ折れ線グラフ]"] は、 link:chart-line-chart-2d.html[折れ線グラフ]の特別な形で、一定期間の数値の変化を表示する場合に便利です。ステップ折れ線グラフの X（水平）軸は一定期間を表します。各値は Y 軸上に一定または不定の時間間隔でプロットされます。

ステップ折れ線グラフは、銀行残高や抵当利息など、定期的または不定期で変化する数字をプロットする場合に役立ちます。

グラフが時系列でデータを表示し、経過時間に基づきX軸でポイントをプロットしたい場合は、「時間の連続的スケールにデータを表示」を参照してください。
