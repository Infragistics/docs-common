= 立体エリア グラフについて

image::images/Chart_About_3D_Area_Chart_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[立体エリア グラフ]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[立体エリア グラフ]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[立体エリア グラフ]"] は、一定期間の変化の量を強調する、または複数の項目を比較します。立体エリア グラフは、プロットされた値の合計を表示することによって、全体に対する各部分の関係も表示します。

エリア グラフは、折れ線グラフの 1 形式ですが、x 軸（水平軸）とデータ マーカーを結ぶ線の間のエリアを色で塗りつぶします。これにより、複数のデータ系列で取り囲まれてデータ点が重なっている個所がはっきりわかります。
