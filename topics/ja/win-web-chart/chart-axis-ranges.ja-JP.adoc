= 軸範囲

数値軸では、軸範囲は軸の先頭から末尾まで数値が異なります。範囲の最小値は、軸の最も低い点の値です。範囲の最大値は、軸の最も高い点の値です。

デフォルトで、Chart は数値軸（または複数の数値軸）の範囲としてデータの最小値と最大値を使用します。範囲を手動で指定するには、 pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangetype.html[UltraChart.Axis.n.RangeType]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangetype.html[UltraChart.Axis.n.RangeType]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangetype.html[UltraChart.Axis.n.RangeType]"]  プロパティを Custom に設定して、 pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangemin.html[UltraChart.Axis.n.RangeMin]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangemin.html[UltraChart.Axis.n.RangeMin]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangemin.html[UltraChart.Axis.n.RangeMin]"]  および  pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangemax.html[UltraChart.Axis.n.RangeMax]"]  pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangemax.html[UltraChart.Axis.n.RangeMax]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.resources.appearance.axisappearance~rangemax.html[UltraChart.Axis.n.RangeMax]"]  プロパティの値を設定します。 ここで、 _n_  はコンテキストの軸（X、X2、Y、Y2、Z、または Z2）の文字を示します。

[source,docX-root]
----
UltraChart.Axis.Y.RangeType = AxisRangeType.Custom
UltraChart.Axis.Y.RangeMin = 0
UltraChart.Axis.Y.RangeMax = 75
----
