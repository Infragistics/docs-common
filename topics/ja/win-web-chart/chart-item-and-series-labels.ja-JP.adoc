= 項目およびシリーズ ラベル

SetLabelAxis オブジェクトで、データ項目はシリーズにグループ化されます。通常、このようにグループ化される場合は、データ行がシリーズを表し、その行の各列の値がそのシリーズの項目を表します。

この結果、項目ラベルが各データ項目（列）に表示され、シリーズ ラベルがグループ化された SetLabelAxis オブジェクトの項目（行）の各グループに表示されます。
