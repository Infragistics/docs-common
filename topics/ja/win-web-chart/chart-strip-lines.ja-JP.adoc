= ストリップ ライン

ストリップ ラインは平面チャートの機能です。開発者は、水平または垂直方向の帯域をデータの後ろに一定間隔で容易に適用できます。これらの「縞」は、グラフによって提示された情報を理解し易くする、視覚的な手掛かりをユーザーに提供します。

たとえば、財務グラフの背後の隔週バンドは、十字線や他の手動操作を使用しなくても、グラフ中の各価格の時期を視角的に判断できる強化された方法を株式仲買人に提供します。このわずかな手掛かりがユーザーの意思決定プロセスのスピードを向上させます。これは、スピード重視のアプリケーションで使用するグラフには重要なことです。

* 塗り潰し

* テクスチャ（画像背景）

これらのプロパティは、デザインタイムにカスタム プロパティ エディタで簡単に設定できます。このエディタではさまざまな選択の予測される外観のプレビュー表示が開発者に示されます。また、ランタイムにプログラムによって動的に変更することもできます。

ストリップ ラインは、グラフに表示された軸の種類に基づいて、素早く適用されます。連続した時間軸上では、ストリップ ラインは一定間隔で並んだグラフ要素の背後に縞を自動的に入れます。データセット別の軸（ほとんどの棒グラフと柱状グラフ）上では、ストリップ ラインは瞬時に復帰して、データセットの棒や柱状のグループごとに、その背後に一定間隔で単一の縞を作成します。これは、より穏やかな提示が必要な場合にユーザーが色の長蛇の列に惑わされるのを防ぎます。
