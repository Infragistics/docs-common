= パーセント値を使用して WebChart の高さと幅を変更

pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.renderingtype.html[RenderingType]"]  pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.renderingtype.html[RenderingType]"]  プロパティが SVG または Flash に設定されると、コントロールの高さと幅をパーセントで設定できます。Height および Width プロパティをパーセント値に設定して、結果を確認してください。画像のレンダリングと異なり、これらのレンダリングの種類はパーセントの高さと幅を認識して、WebChart はクライアントで動的にスケールします。

以下のコードの例は高さと幅をパーセントで設定した WebChart を示します。

[source,docX-root]
----
<table border="1" >
	<tr >
		<td >
			<igchart:UltraChart runat="server" id="UltraChart1" Height="100%" Width="100%">
				<DeploymentScenario RenderingType="Flash" />
			</igchart:UltraChart>
		</td>
		<td>
		</td>
	</tr>
	<tr >
		<td>
		</td>
		<td >
			<igchart:UltraChart runat="server" id="UltraChart2" Height="100%" Width="100%">
				<DeploymentScenario RenderingType="Svg" />
			</igchart:UltraChart>
		</td>
	</tr>
</table>
----
