= 合成チャートの軸要件

以下の表は、 link:chart-composite-chart.html[合成チャート]のレイヤとして使用されるチャートでの各軸の要件の概要を述べます。要件が満たされない場合、合成チャートは描画されません。 

合成チャートの作成方法については、 link:chart-creating-a-composite-chart.html[「合成チャートを作成」]を参照してください。

[NOTE]
====
*注：* 「Numeric」または「Time」の DataType を持つ軸は、複数のグラフ レイヤによって共有されます。「String」の DataType を持つ軸は共有できません。
====
