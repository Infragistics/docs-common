= 平面積層型柱状グラフ データでの作業

このトピックでは、平面積層型柱状グラフでデータが適切に描画されることを保証する支援となる役に立つ情報を説明します。

== データ要件

Chart コントロールによって固有のカスタム データにチャートを簡単にポイントすることができますが、そのチャートが必要とするデータの適切な量とタイプを提供することが重要です。使用しているチャートのタイプに基づいた最小要件をデータが満たさないと、エラーが生成されます。

以下は、平面積層型柱状グラフのデータ要件のリストです。

* データ セットには、ひとつまたは複数の数値列が含まれます。
* 列に含まれる数値は、すべて正か、すべて負のいずれかである必要があります。True でなければエラー メッセージが表示されます。同じ行のすべての列は「積み上げられて」、1 本の縦棒になります。
* 平面積層型柱状グラフを Series オブジェクトにバインドする場合には、Series バインディング要件の詳細は「Series バインディングの要件」を参照してください。
* link:chart-composite-chart.html[合成チャート]で平面積層型柱状グラフを使用する場合には、チャートの軸要件の詳細は、 link:chart-axis-requirements-for-composite-charts.html[「合成チャートの軸要件」]を参照してください。

[NOTE]
====
*注：* 使用可能なデータが、各列がひとつの列セグメントを表し、各行に各列セグメントの点が含まれる書式の場合には、行と列を交換する必要があります。これを実行する詳細は、 link:chart-swap-rows-and-columns.html[「行と列の入れ替え」]を参照してください。
====

== データを平面積層型柱状グラフにマップ

チャート データは以下の規則を使用して描画されます。

* 各行は、複数のセクションに分割されている 1 本の柱状つまりスタックを表します。
* 行が複数ある場合、各行が積層型の柱状をそれぞれ構成します。
各行の各列値は 1 本の積層型柱状の 1 セクションを表します。

描画された平面積層型柱状グラフとともに、サンプルのデータ セットは以下を参照してください。
