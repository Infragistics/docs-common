= XML ファイルへのバインディング

Chart コントロールをバインドできるデータ ソースのひとつは XML ファイルです。Chart コントロールを XML データにバインドするには、XML は ADO.NET データセットのスキーマ要件に従う必要があり、XML は DataSet.ReadXml メソッドを使用して読み取られる機能を持つ必要があります。

=== Chart を XML ファイルにバインドするには、次の手順に従ってください。

[start=1]
. XML をコピーして（このトピックのの最後にある XML データを確認してください）、XML ファイルに貼り付けます。
[start=2]
. 以下の例のコードを使用して、Chart コントロールを XML ファイルにバインドします。

[NOTE]
====
*注：* ファイルへのフル パスを形成する追加のコードが右手にあります。
====

*Visual Basic の場合：*

----
Private Sub Binding_to_an_XML_File_Load(ByVal sender As System.Object, _
  ByVal e As System.EventArgs) Handles MyBase.Load
	Me.UltraChart1.DataSource = "CandleData.xml"
End Sub
----

*C# の場合：*

----
private void Binding_to_an_XML_File_Load(object sender, System.EventArgs e)
{
	this.ultraChart1.DataSource = "CandleData.xml";
}
----

上記のコード例で使用される以下の xml ファイルは、 link:chart-sample-xml-file-candledata.html[ここ]で使用できます。
