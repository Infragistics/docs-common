= 数値、セットラベル、または日付/時刻の軸

各軸は「数値」、「セット ラベル」、または「日付/時刻」軸です。数値軸は、リニアまたは対数目盛をとることができる数値目盛に基づきます。SetLabelAxis オブジェクトは、オプションでシリーズまたは行でグループ化される、均等に間隔が空けられた一連のデータ項目です。日付/時刻軸は、連続または不連続の時刻の目盛に基づきます。
