= 散布図について

image::images/Chart_About_Scatter_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[散布図]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[散布図]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[散布図]"] は、複数の異なるデータのシリーズ内の項目間の関係を示すか、または 2 つの値セットを x/y 座標の 1 シリーズとしてプロットする場合に使用します。散布図では、データの不均等な散らばり方やデータの塊に注意が向けられます。この種類のグラフは、科学データのプロットによく用いられ、予測結果からの収集データの偏差を強調表示できます。

チャートがデータを時系列で表示して、経過時間に従って x 軸で点をプロットするようにしたい場合には、 link:chart-display-data-on-a-time-scale-axis.html[「時間のスケール軸でデータを表示」]を参照してください。
