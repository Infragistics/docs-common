= ポーラ チャートについて

image::images/Chart_About_Polar_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ポーラ チャート]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ポーラ チャート]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ポーラ チャート]"] は、線軸と半径軸を使用して描画する 2 次元のデータ表現です。最も単純な場合、チャートの中心からの距離と方位角度を表す {angle, radial-distance} の 2 つの値をチャート化します。その他のバリエーションは、この基本表現の特殊なケースです。

たとえば、{angle, radial-distance} で表された各データ点をつなぎ、線や欠損値を推定したスプライン補間曲線でつないだ各点や各線を結合できます。別の特殊ケースでは、データ点をグループ化し、チャートを {group, angle, radial-distance} の 3 値の形で表示できます。グループとしてプロットされるデータを、半径軸上に散布点としてプロットしたり、グループ内の各点をつなげてひとつのシリーズを形成することもできます。これらの基本特徴により、ポーラ チャートは半径マッピングによる散布図を効果的にプロットできます。
