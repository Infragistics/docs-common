= Treemap グラフについて

ツリーマップ グラフは、四角形の領域に大きい階層のデータ セットを表示するために使用されます。このグラフは領域に分割され、次に各領域は階層レベルごとに再分割されます。領域のサイズと色によってツリーマップ上のデータを識別できます。

ツリーマップ グラフは多くのディレクトリ レベルに数千のファイルが格納されているハードドライブのコンテンツを視覚化するために通常使用されます。
