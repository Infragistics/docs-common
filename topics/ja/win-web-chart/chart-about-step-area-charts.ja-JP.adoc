= ステップ エリア グラフについて

image::images/Chart_About_Step_Area_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ステップ エリア グラフ]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ステップ エリア グラフ]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ステップ エリア グラフ]"] は、 link:chart-area-chart-2d.html[エリア グラフ]の特別な形で、一定期間の数値の変化を表示する場合に便利です。ステップ エリア グラフの X（水平）軸は一定期間を表します。各値は Y 軸上に一定または不定の時間間隔でプロットされます。ステップ エリア グラフはステップ折れ線グラフと似ていますが、ステップ エリア グラフではデータ シリーズで占めたエリアを色で塗りつぶす点が異なります。

ステップ エリア グラフは、銀行残高や抵当利息など、定期的または不定期で変化する数字をプロットする場合に役立ちます。
