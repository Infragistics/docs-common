= 合成チャート データでの作業

合成チャートの設定で最も注意しなければならないことは、各チャート レイヤのさまざまな要件を満たすことです。このために、 link:chart-series-collection.html[「Series コレクション」]の表を参照してください。この表は、各チャート レイヤごとの軸とデータの要件の概略を示しており、新しい合成チャートを作成する、または既存の合成チャートに新しいチャート レイヤを追加するときに適切な参照先となります。
