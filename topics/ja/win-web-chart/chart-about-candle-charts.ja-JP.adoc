= ローソク足チャートについて

image::images/Chart_About_Candle_Charts_01.png[]

pick:[win-forms="link:{ApiPlatform}win.ultrawinchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ローソク足]"]pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ローソク足]"]pick:[aspnet-old="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.ultrachart.shared.styles.charttype.html[ローソク足]"] （あるいは同時グラフ）は、投資業界で幅広く用いられています。これらのグラフの作成に必要なデータは、より一般的に使用される横棒グラフや縦棒グラフなどのデータと同じであるため、使い方が簡単で既存のグラフシステムに容易に統合できます。

ローソク足チャートは、「ローソク」と呼ばれる視覚エレメントで構成されます。各ローソクはさらに本体と芯の 2 つの部分から成ります。本体の長さで、セッションの始まり値と終わり値の差を示し、本体の色で、始まり値が終わり値より高値か安値かを示します。芯の部分は、セッション中に取引した特定の投資の最高値と最安値を決定します(下図を参照）。

棒グラフでは、リバーサル シグナルに数週間かかる場合が多いのに対し、ローソク足は、複数のセッションでリバーサル シグナルを発信できるので、従来のチャートより早くマーケット ターンを示すことができます。したがって、ローソク足によるマーケット ターンはしばしば、従来のインジケータより進んでいます。

image::images/Chart_Candle_Chart_01.png[]

ローソク行の主要部分は本体です。長さは、投資のセッションの開始値と終了値の範囲を示します。終了値が開始値よりも小さい場合、本体は赤、黒、または暗い色になります。逆に、終了値が開始値よりも大きい場合、本体は通常、白または明るい色になります。本体の上下の行は芯と呼ばれます。上の芯はセッション中の最高値で、下の芯は最安値です。したがって、色、ローソクの本体、そして本体と芯の長さに基づいて、特定の投資に対する市場の傾向を判断することができます。

image::images/Chart_Candle_Chart_02.png[]

たとえば、長く明るい色のローソク本体は、投資株価が上昇していることを意味し、買い傾向です。これに対して、長く暗いローソク本体は、投資株価が下降していることを意味し、売り傾向です。小さい本体（暗いまたは明るい）は、市場の傾向が勢いを失っていることを示します。移動平均は、特定の投資株価が上または下の方向に移動しているかどうかを知る優れた手掛かりを提供します。これは、直近の n 日の価格の平均で計算されます。時間スケールで日が経過すれば、最も古い日が削除され、現在の日が新しい平均計算のために追加されます。傾向の変化は、異なる期間の 2 つの移動平均を交差させると推測できます。これらのシナリオのチャートは、複数の移動平均を持つことができ、移動平均が、その日の高値、安値、開始値、または終了値で計算されるかどうかにおいて柔軟性があります。以下のグラフは、2～5 日の期間の移動平均を示しています。各平均は、株価の終了値に基づいて計算されます。
