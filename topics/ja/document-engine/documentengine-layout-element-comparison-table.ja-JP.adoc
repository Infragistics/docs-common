= レイアウト要素比較表

「レポート レイアウト」のセクションで説明したレイアウト エレメントを使用しなければ、レポートを作成することはできません。これらのエレメントは、その他のレイアウト エレメントだけでなく、その他のコンテンツを含みます。レイアウトエレメントはネストされることも可能で、これにより全く独特なカスタムのレポート作成できます。

レイアウト エレメントの数と各エレメントが実行可能な機能が膨大であるため、以下の表を参照すると、どのレイアウト エレメントがユーザーのシナリオに適しているかを判断する際に役に立ちます。各表の一番上には、ヘルプの「レポート レイアウト」のセクションで詳細に検討されたレイアウトエレメントが配置されており、表の左側には、各レイアウト エレメントが利用できるエレメント、コンテンツ、または追加の設定が縦に配置されています。レイアウト エレメントのいくつかはそのエレメント自身に追加することができることが分かります。
