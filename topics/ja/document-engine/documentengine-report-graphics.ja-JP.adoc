= レポート グラフィックス

レポート グラフィックスは、Brushes、Colors、Fonts、および Images などのいくつかのグラフィックス エレメントによってレポートを生き生きとさせます。レポート グラフィックスを使用して、レイアウト エレメントを強化します。たとえば、ブラシとペンを使用して、エレメントの背景と境界線にそれぞれ色を付けます。Tables、Lists、Text などの link:documentengine-pattern-content.html[パターン コンテンツ]で色を追加してフォントを変更することも可能です。任意のレイアウトまたはコンテンツ エレメントはレポート グラフィックスによって何らかの影響を受ける場合があります。

レポート グラフィックスの詳細は以下のリンクをクリックしてください。

* link:documentengine-brushes.html[ブラシ]
* link:documentengine-canvas.html[キャンバス]
* link:documentengine-colors.html[色]
* link:documentengine-fonts.html[フォント]
* link:documentengine-pens.html[ペン]
* link:documentengine-shapes.html[形状]
