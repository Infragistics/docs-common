= パターン コンテンツ

パターン コンテンツにはパターンを使用することによってすべてスタイル可能なエレメントが含まれています。これらのパターンは、パターン オブジェクトを作成して、次に関連付けられたエレメントに適用することによって、スタイルをグリッド、表、ツリーに適用します。これらのエレメントはほとんどの場合、エレメントを埋めるデータのソースに依存します。

使用可能なパターン コンテンツについては、以下のリストをクリックしてください。

* link:documentengine-grids.html[グリッド]
* link:documentengine-lists.html[リスト]
* link:documentengine-tables.html[表]
* link:documentengine-text.html[テキスト]
* link:documentengine-trees.html[ツリー]
