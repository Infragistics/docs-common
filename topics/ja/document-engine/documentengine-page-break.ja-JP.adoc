= 改ページ

Page Break エレメントは、視覚的ではないエレメントで、関連付けられたレイアウト エレメント内にあるすべての残りのコンテンツを次ページに移動するエレメントです。Page Break エレメントは Microsoft® Word® のページ区切りに非常に似ています。関連付けられている AddPageBreak メソッドを呼び出すことによってページ区切りを以下のエレメントに追加できます。

* Band
* Condition
* Container
* Flow
* Group
* Section
* Segment
