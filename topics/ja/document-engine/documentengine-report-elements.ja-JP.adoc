= レポート要素

レポート エレメントを使用して、コンテンツをレポートに追加してそのコンテンツを論理的フローに配置します。

* link:documentengine-report-layout.html[レポート レイアウト] -- ほとんどすべてのコンテンツのレイアウトに役に立つ Band エレメントから、ページの下部にコンテンツを引き伸ばすためだけに使用される Stretcher エレメントまで、コンテンツ フローを制御するいくつかの役に立つエレメントがここにあります。
* link:documentengine-pattern-content.html[パターン コンテンツ] -- パターン コンテンツは、Grids（行の後ろに行が続く）、Lists（項目の後ろに項目が続く）、Trees（ノードの後ろにノードが続く）など独特な繰り返し可能なコンテンツを作成するエレメントのグループです。パターンを使用してこれらのパターンをエレメント自体に適用することによって、またはスタイル パターンを制御しやすくするためにはエレメント内の個々の項目を適用することによって、セクション内の各エレメントをスタイルできます。
* link:documentengine-quick-content.html[クイック コンテンツ] -- コンテンツを素早く挿入するために必要なのがシンプルなコンテンツ エレメントだけの場合、クイック コンテンツ エレメントが最適です。
* link:documentengine-navigation-helpers.html[ナビゲーション ヘルパー] -- 500 ページのレポートをめくって、Table of Contents、Index、または Bookmarks さえ使用せずに探しているものを見つけ出すことは困難です。
