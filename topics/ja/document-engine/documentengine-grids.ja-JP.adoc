= グリッド

Grid 要素は行と列のレイアウトでコンテンツを表示します。行と列をいくつ追加するかによって、グリッドの各セルがレイアウトにバインドされます。たとえば、セルの幅は列の幅によって決定され、その高さは行の高さによって決定されます。

For i As Integer = 0 To 4
