[[Steps]]
=== 手順

以下では {FinancialChartName} コントロールをページに追加するために必要な手順を示します。

=== 1. {FinancialChartName} コントロールをページに追加します。

以下のコードは、表示に必要な最小限のコードとプロパティ設定で {FinancialChartName} コントロールを追加します。

ifdef::xaml[]
*XAML の場合:*
[source,xaml]
----
<Grid x:Name="LayoutRoot">
 <ig:{FinancialChartName} ItemsSource="{Binding DataCollection}" />
</Grid>
----
endif::xaml[]

ifdef::wpf[]
*Visual Basic の場合:*
[source,vb]
----
Dim financialChart As New {FinancialChartName}()
financialChart.ItemsSource = data
Me.LayoutRoot.Children.Add(financialChart)  
----

*C# の場合:*
[source,csharp]
----
var financialChart = new {FinancialChartName}();
financialChart.ItemsSource = data;
this.LayoutRoot.Children.Add(financialChart);  
----
endif::wpf[]

ifdef::jquery[]
{FinancialChartName} にアタッチする要素を追加します。


*HTML の場合:*
[souce,html]
----
<body>
    <div id="financialChart"></div>
</body>
----

{FinancialChartName} を要素にアタッチします。


*JavaScript の場合:*
[source,javascript]
----
$("#financialChart").{FinancialChartName}({ dataSource: data });
----
endif::jquery[]

=== 2. 結果を確認します。

結果を確認するために、プロジェクトをビルドおよび実行します。手順を正しく実装した場合、{FinancialChartName} は上記のプレビュー セクションで示したように表示されます。
