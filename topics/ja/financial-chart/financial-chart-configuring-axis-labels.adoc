= 軸ラベルの構成

link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] コントロールは、チャートの x 軸と y 軸に基づいてチャートの構成、ラベルの書式やスタイルの設定などを詳細に制御することが可能です。デフォルトでは、ラベルを明示的に設定する必要はありません。

=== このトピックの内容

このトピックは、以下のセクションで構成されます。

* <<ConfigureLabelSettings,ラベル設定>> 
* <<ConfigureStyling,スタイル設定>>
* <<AbbreviatingLargeNumbers,大数の省略>>
* <<CodeSnippet, コード スニペット>>
* <<RelatedContent,関連コンテンツ>>

[[AbbreviatingLargeNumbers]]
=== 大数値の省略

link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] コントロールでは、Y 軸ラベルが省略された数値を表示するかどうかを指定できます。長すぎるラベル値の場合、link:{FinancialChartLink}.{FinancialChartName}{ApiProp}yAxisAbbreviateLargeNumbers.html[pick:[wpf="YAxisAbbreviateLargeNumbers プロパティ"]pick:[jquery="yAxisAbbreviateLargeNumbers オプション"]] を true に設定すると、この機能を有効にします。このプロパティはデフォルトで *true* に設定されています。

[[CodeSnippet]]
=== コード スニペット

以下のコード例は、日付設定およびそ他のスタイル プロパティを使用して x 軸ラベルにスタイル設定する方法示します。

ifdef::jquery[]
*JavaScript の場合:*
[source,javascript]
----
$(".selector").{FinancialChartName}({
    dataSource: data,
    xAxisLabelLeftMargin: 14,
    xAxisLabelTextColor: "Green",
    yAxisLabelTextColor: "Blue"
});
----
endif::jquery[]

ifdef::wpf[]
*XAML の場合:*
[source,xaml]
----
<ig:{FinancialChartName} x:Name="chart1"
    XAxisLabelFontSize="16"
    XAxisLabelForeground="Blue"
	XAxisLabelFontWeight="Bold"
    YAxisLabelFontSize="16"
    YAxisLabelForeground="Green"
    YAxisLabelFontWeight="Bold">
</ig:{FinancialChartName}>
----

*C# の場合:*
[source,csharp]
----
var chart1 = new {FinancialChartName}()
chart1.XAxisLabelFontSize = 16;
chart1.XAxisLabelForeground = new SolidColorBrush(Colors.Blue);
chart1.XAxisLabelFontWeight = FontWeights.Bold;
chart1.YAxisLabelFontSize = 16;
chart1.YAxisLabelForeground = new SolidColorBrush(Colors.Green);
chart1.YAxisLabelFontWeight = FontWeights.Bold;
----


*Visual Basic の場合:*
[source,vb]
----
Dim chart1 = New {FinancialChartName}()
chart1.XAxisLabelFontSize = 16
chart1.XAxisLabelForeground = New SolidColorBrush(Colors.Blue)
chart1.XAxisLabelFontWeight = FontWeights.Bold
chart1.YAxisLabelFontSize = 16
chart1.YAxisLabelForeground = New SolidColorBrush(Colors.Green)
chart1.YAxisLabelFontWeight = FontWeights.Bold
----
endif::wpf[]
