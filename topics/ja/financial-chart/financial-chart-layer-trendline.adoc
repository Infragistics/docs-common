= トレンドライン レイヤー

以下の画像は {FinancialChartName} でトレンドライン レイヤーの使用を表示します。トレンドラインにはさまざまなタイプがあります。QuadraticFit を表示するドロップダウンを使用してタイプを変更できます。

pick:[wpf="image::images/financialchart_wpf_layers_trendline.png[]"]
pick:[jquery="image::images/financialchart_jquery_layers_trendline.png[]"]

[options="header", cols="a,a,a"]

|====

|プロパティ名|型|説明

|link:{DomainChartLink}.{DomainChartName}{ApiProp}trendLineType{HtmlApiExtension}[TrendLineType]
|`FinancialChartVolumeType`
|トレンドラインのタイプを取得または設定します。

|link:{DomainChartLink}.{DomainChartName}{ApiProp}trendLineBrushes{HtmlApiExtension}[TrendLineBrushes]
|`BrushCollection`
|トレンドラインの色設定に使用するブラシのパレットを取得または設定します。

|link:{DomainChartLink}.{DomainChartName}{ApiProp}trendLineThickness{HtmlApiExtension}[TrendLineThickness]
ifdef::wpf,win-forms[]
|`double`
endif::wpf,win-forms[]
ifdef::jquery[]
|`number`
endif::jquery[]
|トレンドラインの太さを取得または設定します。

|link:{DomainChartLink}.{DomainChartName}{ApiProp}trendLinePeriod{HtmlApiExtension}[TrendLinePeriod]
ifdef::wpf,win-forms[]
|`int`
endif::wpf,win-forms[]
ifdef::jquery[]
|`number`
endif::jquery[]
|トレンドラインの期間を取得または設定します。

|====

注: このプロパティはすべてのチャート ペインに適用し、同じトレンドライン タイプはすべてのチャート ペインに適用されます。

以下のコードは link:{FinancialChartLink}.{FinancialChartName}{ApiProp}trendLineType{HtmlApiExtension}[TrendLineType] を設定する方法を紹介します。

ifdef::xaml[]
*XAML の場合:*
[source,xaml]
----
<ig:XamFinancialChart ItemsSource="{Binding}"
    TrendLineType="QuinticFit"
    TrendLineBrushes="Green"
    TrendLineThickness="2" />
----
endif::xaml[]

ifdef::wpf[]
*Visual Basic の場合:*
[source,vb]
----
Dim chart as New XamFinancialChart()
chart.TrendLineType = Infragistics.Controls.Charts.TrendLineType.QuinticFit
chart.TrendLineBrushes = new Infragistics.BrushCollection();
Dim brush As SolidColorBrush = new SolidColorBrush()
brush.Color = Colors.Green
chart.TrendLineBrushes.Add(brush)
chart.TrendLineThickness = 2
----

*C# の場合:*
[source,csharp]
----
var chart = new XamFinancialChart();
chart.TrendLineType = Infragistics.Controls.Charts.TrendLineType.QuinticFit;
chart.TrendLineBrushes = new Infragistics.BrushCollection();
SolidColorBrush brush = new SolidColorBrush();
brush.Color = Colors.Green;
chart.TrendLineBrushes.Add(brush);
chart.TrendLineThickness = 2;
----
endif::wpf[]

ifdef::jquery[]
*JavaScript の場合:*
[source,javascript]
----
$("#financialChart").{FinancialChartName}({ 
    dataSource: data,
    trendLineType: "QuinticFit",
    trendLineBrushes: ["Green"],
    trendLineThickness: 2
});
----

[source,js]
----
$("#financialChart").{FinancialChartName}("option", "trendLineType", "QuinticFit");
$("#financialChart").{FinancialChartName}("option", "trendLineBrushes", ["Green"]);
$("#financialChart").{FinancialChartName}("option", "trendLineThickness", 2);
----
endif::jquery[]
