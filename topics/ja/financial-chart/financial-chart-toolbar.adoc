= {FinancialChartName} ツールバー

=== 目的

このトピックは link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] のツールバーの機能を紹介します。

=== このトピックの内容

このトピックは、以下のセクションで構成されます。

* <<Introduction,概要>>
* <<DateRangeSelector,日付範囲のセレクター>>
* <<IndicatorSelector,インジケーター セレクター>>
* <<ChartTypeSelector,チャート タイプ セレクター>>
* <<RelatedTopics,関連コンテンツ>>

[[Introduction]]
== 概要

{FinancialChartName} はコントロールの上に配置されるツールバーを提供します。チャートで表示されるデータの詳細を変更できます。

pick:[wpf="image::images/financialchart_wpf_toolbar.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar.png[]"]

[[DateRangeSelector]]
== 日付範囲のセレクター

ツールバーの日付範囲セレクターはチャートのデータ範囲を複数の範囲プリセットに変更pick:[wpf="、または「開始」および「終了」日付エディターを使用してカスタム範囲を選択"]できます。

ifdef::wpf[]

「開始」および「終了」日付はlink:financial-chart-pane-zoom.html[ズーム ペイン]が変更されると自動的に更新します。

endif::wpf[]

pick:[wpf="image::images/financialchart_wpf_toolbar_daterange.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar_daterange.png[]"]

[[IndicatorSelector]]
== インジケーター セレクター

インジケーター セレクターで {FinancialChartName} に表示されるように複数のファイナンシャル インジケーター、オーバーレイ、トレンドラインとボリューム タイプを選択できます。

pick:[wpf="image::images/financialchart_wpf_toolbar_indicatortype.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar_indicatortype.png[]"]

財務指標は、株価の動きの計測やトレンドを確認するためにトレーダーによって使用されます。これらのインジケーターは、同じ Y 軸を共有しないためlink:financial-chart-pane-price.html[価格ペイン]の下に表示されます。


ifdef::wpf[]

財務指標タイプの詳細については、{DataChartName} のlink:datachart-financial-indicators-overview.html[財務指標]トピックを参照してください。

endif::wpf[]

pick:[wpf="image::images/financialchart_wpf_toolbar_indicatortype.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar_indicatortype.png[]"]

[[OverlaySelector]]
== オーバーレイ タイプ セレクター

財務オーバーレイは、株価の動きの計測やトレンドを確認するためにトレーダーによって使用されます。これらのオーバーレイは、同じ Y 軸を共有する財務物価シリーズの前に表示されます。

ifdef::wpf[]

財務オーバーレイ タイプの詳細については、{DataChartName} のlink:datachart-financial-overlays-overview.html[財務オーバーレイ] トピックを参照してください。

endif::wpf[]

pick:[wpf="image::images/financialchart_wpf_toolbar_overlaytype.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar_overlaytype.png[]"]

[[TrendlineSelector]]
== トレンドライン タイプ セレクター

トレンドラインは、データの近似曲線の履歴を表示します。選択したトレンドラインはlink:financial-chart-layer-trendline.html[トレンドライン レイヤー]で描画されます。

ボリューム タイプでlink:financial-chart-pane-volume.html[ボリューム ペイン]の株価データのボリュームの表示方法を変更できます。None、Column、Line、および Area チャート タイプから選択できます。

[[ChartTypeSelector]]
== チャート タイプ セレクター

{FinancialChartName} はデータをlink:financial-chart-pane-price.html[価格ペイン]で 4 つのチャート タイプとして表示できます。棒、折れ線、ローソク足、および柱状チャート タイプから選択できます。

pick:[wpf="image::images/financialchart_wpf_toolbar_charttype.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar_charttype.png[]"]

[[VolumeTypeSelector]]
== ボリューム タイプ セレクター

ボリューム タイプ セレクターは、link:financial-chart-pane-volume.html[ボリューム ペイン]で出来高を表示します。棒、折れ線、ローソク足、および柱状チャート タイプから選択できます。

pick:[wpf="image::images/financialchart_wpf_toolbar_volumetype.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar_volumetype.png[]"]
