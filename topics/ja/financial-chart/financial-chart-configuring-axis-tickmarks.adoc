= 軸目盛

目盛りは軸にポイントを表示します。スケールに特定の数値ポイント、またはカテゴリ軸に基本データ項目の値を表します。

[[CodeSnippet]]
== コード スニペット

以下のコードスニペットは、x 軸の目盛りの色、長さ、太さを設定します。

ifdef::jquery[]
*JavaScript の場合:*
[source,javascript]
----
$(".selector").{FinancialChartName}({
    xAxisTickLength: 7,
    xAxisTickStrokeThickness: 5,
    xAxisTickStroke: "Blue"
});
----
endif::jquery[]

ifdef::wpf[]
*XAML の場合:*
[source,xaml]
----
<ig:{FinancialChartName}
    XAxisTickLength = "7"
    XAxisTickStrokeThickness = "5"
    XAxisTickStroke = "Blue" />
----

*C# の場合:*
[source,csharp]
----
var chart = new {FinancialChartName}();
chart.XAxisTickLength = 7;
chart.XAxisTickStrokeThickness = 5;
chart.XAxisTickStroke = Brushes.Blue;

layoutRoot.Children.Add(chart);
----

*Visual Basic の場合:*
[source,vb]
----
Dim chart = New {FinancialChartName}()
chart.XAxisTickLength = 7
chart.XAxisTickStrokeThickness = 5
chart.XAxisTickStroke = Brushes.Blue;

layoutRoot.Children.Add(chart)
----
endif::wpf[]
