[[Intro]]
=== 概要

link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] が持っている link:{DataChartLink}.FinancialLegend.html[FinancialLegend] を有効すると項目がツールバーとチャートのプロット領域間に水平に配置され、チャートの各シリーズとオーバーレイに表示されます。

デフォルトで凡例は無効に設定されます。凡例を有効するには {FinancialChartName} の link:{FinancialChartLink}.{FinancialChartName}{ApiProp}IsLegendVisible.html[IsLegendVisible]pick:[wpf=」プロパティ"]pick:[jquery=」オプション"]を使用します。

[[CodeSnippet]]
=== コード スニペット

以下のコードで {FinancialChartName} の凡例を有効するために使用します。

ifdef::wpf[]

*XAML:*
[source, xaml]
----
<ig:{FinancialChartName} x:Name="chart" ItemsSource="{Binding SelectedStock}" IsLegendVisible="True"/>
----

*C#:*
[source, csharp]
----
chart.IsLegendVisible = true;
----

*Visual Basic:*
[source, vb]
----
chart.IsLegendVisible = True
----

endif::wpf[]

ifdef::jquery[]

*In JavaScript:*
[source, javascript]
----
$(function () {
    $("#chart").{FinancialChartName}({
        dataSource: stocks,
        isLegendVisible: true
    });
});
----

endif::jquery[]
