= {FinancialChartName} の軸範囲の構成

link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] コントロールで、軸の範囲がデフォルトでデータ ソースに基づいて計算され、以下のように構成できます。範囲の最小値は、軸の最小値です。範囲の最大値は、軸の最大値です。

[[Overview]]
== 概要

デフォルトで、{FinancialChartName} コントロールは、チャート プロット領域を最大化するために、最小データ ポイントおよび最大データ ポイントに基づいて軸の範囲の最小値と最大値を計算します。軸の最小値と最大値の自動計算は、データ ポイントのセットに適切でない場合があります。たとえば、チャートの端とプロット シリーズの間に余分なスペースを表示する場合、{FinancialChartName} 軸範囲を書式設定する必要があります。

{FinancialChartName} コントロールで以下のプロパティを設定して Y 軸範囲を構成できます。

== コード スニペット

以下のサンプル コードは、 {FinancialChartName} コントロールの Y 軸の軸範囲を変更する方法を示します。

ifdef::jquery[]
*JavaScript の場合:*
[source,javascript]
----
$(".selector").{FinancialChartName}({
    dataSource: data,
    xAxisMinimumValue: new Date(2016, 4, 1),
    xAxisMaximumValue: new Date(2017, 11, 1),
    yAxisMinimumValue: 600,
    yAxisMaximumValue: 1000
});
----
endif::jquery[]

ifdef::xaml[]
*XAML の場合:*
[source, xaml]
----
<ig:{FinancialChartName} ItemsSource="{Binding}"
    XAxisMinimumValue="4/1/2016"
    XAxisMaximumValue="11/1/2017"
    YAxisMinimumValue="600"
    YAxisMaximumValue="1000" />
----

*C# の場合:*
[source, csharp]
----
var chart = new {FinancialChartName}();
chart.XAxisMinimumValue = new DateTime(2016, 4, 1);
chart.XAxisMaximumValue = new DateTime(2017, 11, 1);
chart.YAxisMinimumValue = 600;
chart.YAxisMaximumValue = 1000;
----

*VB の場合:*
[source, vb]
----
chart.XAxisMinimumValue = New DateTime(2016, 4, 1)
chart.XAxisMaximumValue = New DateTime(2017, 11, 1)
chart.YAxisMinimumValue = 600
chart.YAxisMaximumValue = 1000
----
endif::wpf[]
