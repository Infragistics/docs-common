= 定義済みの図形の使用

このトピックは、Infragistics Excel Engine における形状のサポートの概要を説明し、使い方のコード スニペットを示します。

このトピックは以下のように構成されています。

* <<One,はじめに>>
* <<Two,サポートされる形状 タイプ>>
* <<Three,サポートされる形状のカスタマイズ>>
* <<Four,関連トピック>>

[[One]]
== はじめに

Microsoft Excel では、ワークシート内で定義済み形状を使用できます。Infragistics Excel Engine は、ワークシートの既存の形状のコレクションにより形状をサポートします。より多くの形状タイプの作成を許可することにより、この機能が既存のサポートに追加されます。
