== 概要

=== ヘッダーの概要

列ヘッダーと行ヘッダーは、列と行を視覚的に特定するために使用します。また、現在選択されているセルやセル領域を強調表示する場合にも使用します。

以下のスクリーンショットは、ワークシートのヘッダーを示します。現在選択されているセルによって、最初の行 (「1」) と最初の列 (「A」) に強調表示されたヘッダーがあります。

== 表示 / 非表示

[[_Hlk368069110]]

=== 概要

列ヘッダーと行のヘッダーは、`ShowRowAndColumnHeaders` プロパティを使用して、表示または非表示を切り替えることができます。

=== プロパティ設定

以下の表は、任意の構成とそれを管理するプロパティ設定のマッピングを示します。

[options="header", cols="a,a,a"]
|====
|目的:|使用するプロパティ:|設定値:

|列ヘッダーと行ヘッダーを表示する
|`ShowRowAndColumnHeaders`
|`True`

|列ヘッダーと行ヘッダーを非表示にする
|`ShowRowAndColumnHeaders`
|`False`

|====

[[_Hlk337817761]]

=== 例

以下のコードは、開かれているワークブックの最初のワークシートで列ヘッダーと行ヘッダーを表示する方法を示します。

ifdef::wpf[]
*Visual Basic の場合:*
[source,vb]
----
Me.spreadsheet1.Workbook.Worksheets(0).DisplayOptions.ShowRowAndColumnHeaders = True
----
endif::wpf[]

*C# の場合:*
[source,csharp]
----
this.spreadsheet1.Workbook.Worksheets[0].DisplayOptions.ShowRowAndColumnHeaders = true;
----
