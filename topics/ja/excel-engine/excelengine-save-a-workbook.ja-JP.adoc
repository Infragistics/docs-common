= ワークブックの保存 ({ExcelEngineName})

link:{ExcelEngineApiLink}infragistics.documents.excel.workbook~save.html[Save] メソッドを呼び出すことによって、 link:{ExcelEngineApiLink}infragistics.documents.excel.workbook.html[Workbook] オブジェクトを保存できます。ただし、保存する時にワークブックは少なくともひとつの表示されるワークシートを含む必要があります。そうでないと、例外がスローされます。

以下のコード例は Workbook オブジェクトを Excel ファイルとして保存する方法を示します。

ifdef::wpf[]
*Visual Basic の場合:*
[source, vb]
----
Dim workbook1 As New Workbook()
Dim worksheet As Worksheet = workbook1.Worksheets.Add("Sheet1")
' 列ヘッダーをいくつか作成します。
worksheet.Rows(1).Cells(1).Value = "Morning"
worksheet.Rows(1).Cells(2).Value = "Afternoon"
worksheet.Rows(1).Cells(3).Value = "Evening"
workbook1.Save("filename.xls")
----
endif::wpf[]

*C# の場合：*
[source, csharp]
----
Workbook workbook1 = new Workbook();
Worksheet worksheet = workbook1.Worksheets.Add("Sheet1");
// 列ヘッダーをいくつか作成します。
worksheet.Rows[1].Cells[1].Value = "Morning";
worksheet.Rows[1].Cells[2].Value = "Afternoon";
worksheet.Rows[1].Cells[3].Value = "Evening";
ifdef::wpf[]
workbook1.Save("filename.xls");
endif::wpf[]
ifdef::xamarin[]
workbook1.Save(outputStream);
endif::xamarin[]
----
