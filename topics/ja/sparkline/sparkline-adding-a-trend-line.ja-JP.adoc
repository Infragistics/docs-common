== 近似曲線の追加

=== 概要

近似曲線は、 _{SparklineName}_   コントロールで、スパークライン レイヤーの上に別のレイヤーとして表示されます。近似曲線を表示するには、 link:{SparklineLink}.{SparklineName}~trendlinetype.html[TrendLineType] プロパティを使用します。

近似曲線は、チャートがバインドされているデータの値を使用して、 link:{SparklineLink}.{SparklineName}~trendlinetype.html[TrendLineType] プロパティで指定されたアルゴリズムにしたがって計算されます。対応する近似曲線は {SparklineName} の概要のトピックでリストし、説明しています。

近似曲線のタイプは一度に 1 つしか表示できません。(複数の近似曲線を同時に表示することはできません。)

既定では、近似曲線は表示されません。

=== 近似曲線タイプの指定

以下の表は、 _{SparklineName}_   近似曲線タイプとそれらを提供する link:{SparklineLink}.{SparklineName}~trendlinetype.html[TrendLineType] プロパティ設定の関係を示しています。

|====
|近似曲線タイプ|TrendLineType プロパティ設定

|指数平均
|ExponentialAverage

|指数フィット

|リニア フィット

|ModifiedAverage

|SimpleAverage

|====

=== コード例

以下のコードは、 link:{SparklineLink}.{SparklineName}~trendlinetype.html[TrendLineType] プロパティの QuinticFit への設定を示しています。

ifdef::sl,wpf,win-phone,win-universal,xamarin[]

*XAML の場合:*

[source,xaml]
----
<igSparkline:{SparklineName} TrendLineType="QuinticFit"/>
----

*C# の場合:*

[source,csharp]
----
this.{SparklineName}1.TrendLineType = SparklineTrendType.QuinticFit;
----

ifdef::sl,wpf,win-phone,win-universal[] 
*Visual Basic の場合:*

[source,vb]
----
Me.{SparklineName}1.TrendLineType = SparklineTrendType.QuinticFit
----
endif::sl,wpf,win-phone,win-universal[] 
endif::sl,wpf,win-phone,win-universal,xamarin[] 

ifdef::win-forms[] 
*C# の場合:*

[source,csharp]
----
this.{SparklineName}1.TrendLineType = TrendLineType.QuinticFit;
----

*Visual Basic の場合:*

[source,vb]
----
Me.{SparklineName}1.TrendLineType = TrendLineType.QuinticFit
----
endif::win-forms[] 

=== 近似曲線の構成

近似曲線は、以下の面で構成できます。

* 近似曲線タイプ
* 近似曲線期間
* 近似曲線の色
* 近似曲線太さ

近似曲線の構成方法の詳細については、 link:sparkline-configuring-the-trend-line.html[近似曲線の構成]のトピックを参照してください。
