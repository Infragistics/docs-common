= 軸ラベルの構成

link:{ShapeChartLink}.{ShapeChartName}.html[{ShapeChartName}] コントロールは、チャートの x 軸と y 軸に基づいてチャートの構成、ラベルの書式やスタイルの設定などを詳細に制御することが可能です。デフォルトでは、ラベルを明示的に設定する必要はありません。シェープ チャートは、データ内で最初の適切な文字列プロパティを使用し、ラベルに使用します。

[[ConfigureLabelSettings]]
=== ラベル設定

{ShapeChartName} コントロールでは、以下のプロパティで x 軸および y 軸のラベルの回転角度、マージン、水平/垂直の配置、不透明度、パディングと表示を変更できます。

[options="header", cols="a,a,a,a"]
|====
|プロパティ名
|プロパティ タイプ
2+|説明

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}xaxislabelangle.html[XAxisLabelAngle] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}yaxislabelangle.html[YAxisLabelAngle] 
|`double`
2+|x 軸と y 軸のラベルの回転角度を決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelHorizontalAlignment.html[XAxisLabelHorizontalAlignment] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelHorizontalAlignment.html[YAxisLabelHorizontalAlignment]
|`HorizontalAlignment`
2+|x 軸と y 軸のラベルの水平方向の配置を決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelVerticalAlignment.html[XAxisLabelVerticalAlignment] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelVerticalAlignment.html[YAxisLabelVerticalAlignment]
|`VerticalAlignment`
2+|x 軸と y 軸のラベルの垂直方向の配置を決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelVisibility.html[XAxisLabelVisibility] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelVisibility.html[YAxisLabelVisibility]
|`Visibility`
2+|x 軸と y 軸のラベルを表示するかどうかを決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelMargin.html[XAxisLabelMargin] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelMargin.html[YAxisLabelMargin]
|`Thickness`
2+|x 軸と y 軸のラベルに適用するマージンを決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelOpacity.html[XAxisLabelOpacity] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelOpacity.html[YAxisLabelOpacity]
|`double`
2+|x 軸または y 軸ラベルの不透明度を決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelPadding.html[XAxisLabelPadding] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelPadding.html[YAxisLabelPadding]
|`Thickness`
2+|x 軸と y 軸のラベルに適用するパディングを決定します。

|====

[[ConfigureStyling]]
=== スタイル設定
シェープ チャートの x 軸および y 軸のラベルのルックアンドフィールをスタイル設定できます。主にフォントタイプ、フォント サイズ、フォントの太さなど異なるフォント スタイルをラベルに適用できます。以下のプロパティを使用します。

[options="header", cols="a,a,a,a"]
|====
|プロパティ名
|プロパティ タイプ 
2+|説明

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelFontFamily.html[XAxisLabelFontFamily] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelFontFamily.html[YAxisLabelFontFamily] 
|`FontFamily`
2+|x 軸と y 軸ラベルに使用するフォント ファミリを決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelFontSize.html[XAxisLabelFontSize] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelFontSize.html[YAxisLabelFontSize]
|`double`
2+|x 軸と y 軸のラベルのフォントサイズを設定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelFontStretch.html[XAxisLabelFontStretch] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelFontStretch.html[YAxisLabelFontStretch]
|`FontStretch`
2+|x 軸と y 軸のラベルに適用するフォント伸縮を決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelFontStyle.html[XAxisLabelFontStyle] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelFontStyle.html[YAxisLabelFontStyle]
|`FontStyle`
2+|x 軸と y 軸のラベルのフォント スタイルを決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelFontWeight.html[XAxisLabelFontWeight] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelFontWeight.html[YAxisLabelFontWeight]
|`FontWeight`
2+|x 軸と y 軸のラベルのフォントの太さを決定します。

|link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabelForeground.html[XAxisLabelForeground] +
link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}YAxisLabelForeground.html[YAxisLabelForeground]
|`Brush`
2+|x 軸と y 軸のラベルのテキストの色を決定します。

|====

[[ConfigureFormat]]
=== 書式

{ShapeChartName} では、軸ラベルは書式を適用しない常にシンプルなテキストを表示します。ただし、link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabel.html[XAxisLabel] プロパティに書式文字列を設定して軸ラベルの書式を変更できます。たとえば、x 軸に沿って通貨データをプロットしている場合、デフォルトのラベルは通貨値の小数点表現を表示するのみです。これらの値を通貨記号と一緒に表示したい場合、少数を指定した番号が後に続く C 書式指定子など link:{ShapeChartLink}.{ShapeChartXY}{ApiProp}XAxisLabel.html[XAxisLabel] プロパティを設定する必要があります。

.NET フレームワーク複合書式文字列の詳細は、以下のオンライン リソースを参照してください。

* link:http://msdn.microsoft.com/ja-jp/library/dwhawy9k.aspx[標準の数値書式文字列]
* link:http://msdn.microsoft.com/ja-jp/library/az4se3k1(v=VS.100).aspx[標準日付/時刻書式の文字列]

.注:
[NOTE]
====
XAML の中括弧 ('{' と '}') はマークアップ拡張のための特殊文字です。文字列の最初の値が中括弧でなければならない場合、中括弧の空のセットをエスケープ シーケンスとして使用する必要があります。これによって、リテラル値として取得すべき中括弧の空のセットにテキストが続くことができます。
====

[[CodeSnippet]]
=== コード スニペット
以下のコード例は、日付設定およびそ他のスタイル プロパティを使用して x 軸ラベルにスタイル設定および書式設定する方法示します。

ifdef::wpf[]
*XAML の場合:*
[source,xaml]
----
<ig:{ShapeChartName} x:Name="chart1"  
                  Title="Financial Data"
                  Subtitle="(High, Average, Low)"
                  XAxisLabelFontSize="16"
                  XAxisLabelMargin="14,0,0,0"
                  XAxisLabel="{}{0:$#.00}"
                  XAxisLabelForeground="Red"
                  XAxisLabelFontWeight="Bold">
</ig:{ShapeChartName}>
----

endif::wpf[]

*C# の場合:*
[source,csharp]
----
chart1.XAxisLabel = "{}{0:$#.00}";
chart1.XAxisLabelFontSize = 16;
chart1.XAxisLabelLeftMargin = new Thickness(14);
chart1.XAxisLabelForeground = new SolidColorBrush(Color.Red);
chart1.XAxisLabelFontWeight = FontWeights.Bold;
----


*Visual Basic の場合:*
[source,vb]
----
chart1.XAxisLabel = "{0:$#.00}"
chart1.XAxisLabelFontSize = 16
chart1.XAxisLabelLeftMargin = 14
chart1.XAxisLabelForeground = New SolidColorBrush(Color.Red)
chart1.XAxisLabelFontWeight = FontStyle.Bold
----


ifdef::wpf[]
以下のコード スニペットのように書式設定文字列を使用して XAxisLabel プロパティを設定した場合、同じ結果になります。

*XAML の場合:*
[source,xaml]
----
<ig:{ShapeChartName} x:Name="chart1"  
                  Title="Financial Data"
                  Subtitle="(High, Average, Low)"
                  XAxisLabel="{}{0:$#.00}"
                  XAxisLabelFontSize="16"
                  XAxisLabelForeground="Red"
                  XAxisLabelFontWeight="Bold">
</ig:{ShapeChartName}>

----
endif::wpf[]
