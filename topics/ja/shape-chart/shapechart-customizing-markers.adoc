= チャート マーカーの構成

マーカーは、link:{ShapeChartLink}.{ShapeChartName}.html[{ShapeChartName}] コントロールのプロット領域のデータ ポイント値を表示する視覚的要素です。マーカーは、値が主グリッド線と副グリッド線の間にある場合に、指定したデータ ポイントの値をただちに識別できるようユーザーをサポートをします。
このセクションは、{ShapeChartName} コントロールのマーカーでの作業に関するタスクベースの手順についての役立つ情報を提供します。

[[supportedcharttypes]]
== サポートされるチャート タイプ

以下のチャート タイプはマーカーをサポートします。

* バブル 
* 折れ線
* ポイント
* 多角形
* スプライン

注: 多角形チャートはポイントによって定義される図形の中央にマーカーを表示します。その他のチャート タイプはデータ項目の X/Y 座標にマーカーを描画します。

以下は、{ShapeChartName} のすべてのシリーズのマーカー タイプを変更するためのコードです。
ifdef::wpf[]
*XAML の場合:*
[source,xaml]
----
<ig:XamShapeChart 
    ItemsSource="{Binding}" 
    ChartType="Spline" 
    MarkerTypes="Triangle" 
</ig:XamShapeChart>
----
endif::wpf[]

ifdef::wpf[]
*C# の場合:*
[source,csharp]
----
this.shapeChart.MarkerTypes = new Infragistics.Controls.Charts.MarkerTypeCollection 
{ 
    MarkerType.Triangle 
};
----

*Visual Basic の場合:*
[source,vb]
----
Me.shapeChart.MarkerTypes = New Infragistics.Controls.Charts.MarkerTypeCollection From 
{
    MarkerType.Triangle
}
----
endif::win-forms[]

[[markerappearance]]
== マーカーの外観

チャート マーカーの外観は、{ShapeChartName} クラスの link:{ShapeChartLink}.{ShapeChartDomain}{ApiProp}MarkerBrushes.html[MarkerBrushes] および link:{ShapeChartLink}.{ShapeChartDomain}{ApiProp}MarkerBrushes.html[MarkerOutlines] プロパティで管理できます。

以下のコード スニペットは、{ShapeChartName} の MarkerBrushes および MarkerOutlines の変更方法を示します。

ifdef::wpf[]
*XAML の場合:*
[source,xaml]
----
<ig:XamShapeChart 
      MarkerBrushes=”Red Blue” 
      MarkerOutlines=”Red Blue”
      ChartType="Line">
</ig:XamCategoryChart>

----



*C# の場合:*
[source,csharp]
----
shapeChart.MarkerBrushes = new BrushCollection 
{ 
    Brushes.Red, 
    Brushes.Blue 
};
shapeChart.MarkerOutlines = new BrushCollection 
{ 
    Brushes.Red, 
    Brushes.Blue 
};
----

*Visual Basic の場合:*
[source,vb]
----
Me.shapeChart.MarkerBrushes = New BrushCollection From 
{
    Brushes.Red, 
    Brushes.Blue
}
Me.shapeChart.MarkerOutlines = New BrushCollection From 
{
    Brushes.Red, 
    Brushes.Blue
}
----
endif::wpf[] 

以下のスクリーンショットは、折れ線チャート タイプでマーカーをカスタマイズした {ShapeChartName} コントロールを示します。
