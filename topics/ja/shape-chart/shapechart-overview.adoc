= 概要

== {ShapeChartName} の概要

xamShapeChart は軽量で高パフォーマンスなチャートです。このチャートは、散布シリーズの表示、更にポリラインおよび多角形のデータ ビジュライゼーションもサポートします。チャートはシェープ ファイルまたは X/Y ポイントの List の List を使用して定義されるカスタム図形のデータをプロットできます。

チャートはスマート データ アダプターを使用してデータを分析し、適切な可視化で描画します。たとえば、ChartType を Auto に設定し、データセットがポイントの `List` の `List` で構成される場合、ポイント チャート、ポリライン チャート、多角形チャートがプロットされます。

ただし、`ChartType` プロパティをサポートされる 9 つの可視化のうちの 1 つに設定してチャート タイプを明示的に指定することもできます。

* ポイント
* バブル
* エリア
* 等高線
* 高密度
* 折れ線
* スプライン
* ポリライン
* 多角形

image::images/shapechart_overview.png[]


また、図形チャートは、FixedCost、VariableCost、SalesRevenue、および SalesUnits データ列を持つデータ項目が 1 つあれば損益分岐点データを描画できます。
