[[Steps]]
== {CategoryChartName} を {PlatformName} アプリケーションへ追加

=== 1. {ShapeChartName} をページに追加

以下のコードは、表示に必要な最小限のコードとプロパティ設定で {ShapeChartName} コントロールを追加します。以下のコード はlink:resources-sample-scatter-data.html[サンプル散布データ]のリソースを使用しています。

ifdef::wpf[]
*XAML の場合:*
[source,xaml]
----
<Grid x:Name="layoutRoot">
    <ig:{ShapeChartName} ItemsSource="{Binding DataCollection}" />
</Grid>
----
endif::wpf[]

*C# の場合:*
[source,csharp]
----
var data = new ScatterDataSource();
var shapeChart = new {ShapeChartName}();
shapeChart.ItemsSource = data;
this.layoutRoot.Children.Add(shapeChart);
----

ifdef::wpf,win-forms[]
*VB の場合：*
[source,vb]
----
Dim data As New ScatterDataSource()
Dim shapeChart As New {ShapeChartName}()
shapeChart.ItemsSource = data
Me.layoutRoot.Children.Add(shapeChart)
----
endif::wpf,win-forms[]


=== 2. 結果を確認します。

結果を確認するために、プロジェクトをビルドおよび実行します。手順を正しく実装した場合、{ShapeChartName} は上記のプレビュー セクションで示したように表示されます。
