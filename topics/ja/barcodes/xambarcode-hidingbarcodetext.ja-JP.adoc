== トピックの概要

=== 目的

人間が認識できるテキストを非表示にするために {Barcode128Name} コントロールを使用したデモ。

== 人間が認識できるテキストを非表示 – 概念的概要

=== 概要

{Barcode128Name} はブール型 link:{BarcodesLink}.{Barcode128Name}{ApiProp}showtext.html[ShowText] プロパティを公開します。このプロパティをデフォルト値である  _true_   に設定すると、人間が認識できるテキストまたはバーコードのデータは、バーコードの下で表示されます。このプロパティは、人間が認識できるテキストが画面に描画されることを停止するには  _false_   に設定する必要があります。

== バーコード テキストを非表示 – コード例

=== 説明

以下のコード例は、{Barcode128Name} の link:{BarcodesLink}.{Barcode128Name}{ApiProp}showtext.html[ShowText] プロパティを使用して、人間が認識できるバーコード テキストを表示または非表示にする方法を示します。デフォルトでは、このプロパティは  _true_   に設定され、テキストを表示します。このプロパティは、人間が認識できるテキストを非表示にするには、 _false_   に設定する必要があります。

=== コード

ifdef::xaml[]

*XAML の場合:*

[source,xaml]
----
<ig:XamGs1DataBarBarcode 
    CodeType="Expanded"
    Data="1234567890"
    ShowText=" />
----

endif::xaml[]

ifdef::xaml,win-forms[]

*C# の場合:*

ifdef::xaml[]
----
XamGs1DataBarBarcode   
Barcode = new XamGs1DataBarBarcode;
Barcode.Data = "1234567890";
Barcode.CodeType = GS1CodeType.Expanded;
Barcode.ShowText = false;
----
endif::xaml[]

ifdef::win-forms[]
----  
UltraCode128Barcode Barcode = new UltraCode128Barcode();
Barcode.Data = "1234567890";
Barcode.CodeType = Code128CodeType.Standard;
Barcode.ShowText = false;
----
endif::win-forms[]

endif::xaml,win-forms[]

ifdef::xaml,win-forms[]

*Visual Basic の場合:*

ifdef::xaml[]
----
Dim Barcode As New  XamGs1DataBarBarcode()
Barcode.Data = "1234567890"
Barcode.CodeType = GS1CodeType.Expanded   
Barcode.ShowText = false
----
endif::xaml[]

ifdef::win-forms[]
----
Dim Barcode As New UltraCode128Barcode()
Barcode.Data = "1234567890"
Barcode.CodeType = Code128CodeType.Standard 
Barcode.ShowText = False
----
endif::win-forms[]

endif::xaml,win-forms[]
