= XDimension の構成

link:{BarcodesLink}.{BarcodesBase}{ApiProp}xdimension.html[XDimension] は、( pick:[wpf,win-forms="MaxiCode を除き、"] すべてのバーコード グリッドの記号に対して) 記号の狭い要素幅をミリメートル単位で定義します。このプロパティ値は、Stretch.None を使用するときに記号の幅を指定するために使用します。各バーコードの最小値は、デフォルト値として設定されます。

pick:[xaml,win-forms="*2 つのバーコード間の比較です。*"]

ifdef::wpf,win-forms[]
image::images/xamBarcode_XDimension_01.png[]
endif::wpf,win-forms[]

ifdef::xamarin[]
image::images/xamBarcode_XDimensionQR_01.png[]
endif::xamarin[]

ifdef::wpf[]

*XAML の場合:*
[source,xaml]
----
<ig:{Barcode128Name} x:Name="Left" Data="Code Data" Stretch="None" XDimension="0.25" />
<ig:{Barcode128Name} x:Name="Right" Data="Code Data" Stretch="None" XDimension="0.8"/>
----

endif::wpf[]

ifdef::xamarin[]

*XAML の場合:*
[source,xaml]
----
<ig:XamQRCodeBarcode x:Name="barcode1" Data="Code Data" Stretch="None" XDimension="2" />
<ig:XamQRCodeBarcode x:Name="barcode2" Data="Code Data" Stretch="None" XDimension="4"/>
----

endif::xamarin[]

ifdef::wpf,win-forms[]

*Visual Basic の場合:*
[source,vb]
----
Dim barcode1 As New {Barcode128Name}()
barcode1.Data = "Code Data"
barcode1.Stretch = Stretch.None
barcode1.XDimension = 0.25

Dim barcode2 As New {Barcode128Name}()
barcode2.Data = "Code Data"
barcode2.Stretch = Stretch.None
barcode2.XDimension = 0.8
----

*C# の場合:*
[source,csharp]
----
var barcode1 = new {Barcode128Name}();
barcode1.Data = "Code Data";
barcode1.Stretch = Stretch.None;
barcode1.XDimension = 0.25;

var barcode2 = new {Barcode128Name}();
barcode2.Data = "Code Data";
barcode2.Stretch = Stretch.None;
barcode2.XDimension = 0.8;
----

endif::wpf,win-forms[]

ifdef::xamarin[]

*C# の場合:*

----
var barcode1 = new XamQRCodeBarcode();
barcode1.Data = "Code Data";
barcode1.Stretch = BarcodeStretch.None;
barcode1.XDimension = 2;

var barcode2 = new XamQRCodeBarcode();
barcode2.Data = "Code Data";
barcode2.Stretch = BarcodeStretch.None;
barcode2.XDimension = 4;
----

endif::xamarin[]

ifdef::android[]

*Java の場合:*

----
{Barcode128Name} barcode1 = new {Barcode128Name};
barcode1.setData("Code Data");
barcode1.setStretch(Stretch.None);
barcode1.setXDimension("0.25");

{Barcode128Name} barcode2 = new {Barcode128Name};
barcode2.setData("Code Data");
barcode2.setStretch(Stretch.None);
barcode2.setXDimension("0.8");
----

endif::android[]
