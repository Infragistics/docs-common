[[WhyUse]]
== コントロール コンフィギュレーターの利点

コントロール コンフィギュレーターの機能:

- ターゲット コントロールを構成。
- ギャラリー アイテムを使用してコントロールの外観をプレビューを確認しながら構成。
- 機能ごとに分類されたプロパティ グリッドを使用して高機能な Infragistics {PlatformName} コントロールの機能を最大限に活用できます。

コントロール コンフィギュレーターを使用することにより、高機能な Infragistics {PlatformName} コントロールをすばやく使いこなすことができるようになります。

[[Features]]
== 機能

機能はすべてのコントロール コンフィギュレーター共通です。

- クイック セット (ゲージのみ) は、クイック セット リボン ギャラリーから定義済みのコントロールを選択するだけでコントロールをすばやく構成できます。
- ブラシとアウトラインは、ブラシまたはアウトラインのセットを設定できます。多数の色、ブラシ、アウトラインは適用先のターゲットによって動的に決定されます。4 が必要な場合、4 が表示されます。7 の場合は 7 が表示されます。 
- コントロール コンフィギュレーターでは、ランタイムで描画される背景色と同じ色を使用してコントロールを構成できます。
- ギャラリーのブラシやアウトラインをコンフィギュレーターの背景色を使用して描画することもできます。背景色に対してブラシがどのように表示されるかを確認できます。
- コントロールの境界ボックス アウトラインは、親レイアウト コントロールによって提供される領域に相対するコントロール サイズを確認できます。
- データ バインディング ビルダーはデータ バインディングを作成するビジュアル ツールです。
- デザイン タイム データは、コントロールが設定した値に対してどう反応するかを確認できます。
- リソース セレクターは、App.xaml に定義されたリソースの割り当て、またはコントロールのプロパティにローカル ページを割り当てます。
- 高機能プロパティ グリッドはデータ バインドの設定、テスト、リソースの設定、プロパティ値の再設定などが可能です。
- リボン ボタンは {PlatformName} コントロールおよびヘルプ ドキュメントへリンクします。
