[[Introduction]]
== 概要 

円チャート コントロール コンフィギュレーターは、Infragistics {PlatformName} {PieChartName} コントロールを UI ですばやく構成し、リアルタイムでプレビューが可能なビジュアル ツールです。プレビューは、モバイル デバイスのランタイム時の外観とほとんど同じです。

ItemsSource へのデータ バインディングが設定した後、リアルタイム サンプル データが作成されます。このサンプル データは、プロジェクトの型に基づいてコンフィギュレーター レイアウト サーフェイスで円チャートを描画します。

[[Features]]
== 機能
  
以下の機能は {PieChartName} のコントロール コンフィギュレーターで利用可能です。

- ユーザー型に基づくリアルタイム サンプル データは円チャートで描画されます。
- ギャラリー項目を選択すると、操作、レイアウト、またはラベルを変更します。
- サンプル データを変更し、デザイン タイム エクスペリエンスを向上します。
- コントロールの凡例を追加または削除できます。
