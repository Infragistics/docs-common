[[Introduction]]
== 概要  

データ グリッド コントロール コンフィギュレーターは、Infragistics Xamarin.Forms {DataGridName} コントロールを UI ですばやく構成し、リアルタイムでプレビューが可能なビジュアル ツールです。プレビューは、モバイル デバイスのランタイム時の外観とほとんど同じです。

ItemsSource へのデータ バインディングを設定後、リアルタイムでサンプル データが作成されます。このサンプル データは、プロジェクト タイプに基づいてコンフィギュレーター レイアウト サーフェイスにデータ グリッドを描画します。

[[Features]]
== 機能

以下の機能は {DataGridName} のコントロール コンフィギュレーターで利用可能です。

- 列の自動生成または手動生成の設定
- 手動生成の場合、表示する列の選択
- 表示される行の数の設定
- ヘッダー クリックでの並べ替え操作の定義
- 行選択の設定 (単一、複数、なし)
- 列アニメーションの設定
