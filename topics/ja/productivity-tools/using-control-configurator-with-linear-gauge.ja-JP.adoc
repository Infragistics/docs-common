[[Features]]
== 機能

以下の機能は {LinearGaugeName} のコントロール コンフィギュレーターで利用可能です。

- 範囲の作成、削除、変更。
- デザイン タイムとランタイムでコントロールの変更をアニメーション化。
- ギャラリー アイテムを選択して図形の範囲、スケール、針を変更。
- コントロールの最大値と最小値。
- Min または Max 値が変更した場合の範囲値を動的に構成。
