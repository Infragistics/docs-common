== 概要

=== {DoughnutChartName} 概要

{DoughnutChartName} コントロールは {PieChartName} と同様、変数の発生を比例的に表します。{DoughnutChartName} は、複数の変数をコンセントリック リングで表示でき、pick:[wpf,win-forms="階層データの視覚化"]pick:[xamarin="ラベル、凡例、選択と展開"]を組み込みでサポートします。

ifdef::wpf,win-forms[]
.注:
[NOTE]
====
現在、階層データのサポートは community technology preview (CTP) としてリリースされています。
====
endif::wpf,win-forms[]

== 主要機能

=== 主要な機能の概要表

以下の表に、 _{DoughnutChartName}_   コントロールの主要機能をまとめました。

|====
|機能|説明

|複数リング サポート
|_{DoughnutChartName}_ は、複数のリングを、異なるコレクションにバインド可能な各リングと同時に表示できます。または共通のデータ ソースを共有できます。

|_{DoughnutChartName}_ は、1 つ以上のスライスの状態を `選択された状態`に設定する API を公開します。任意で、単一のカスタム ビジュアル スタイルを選択済みスライスに適用できます。
