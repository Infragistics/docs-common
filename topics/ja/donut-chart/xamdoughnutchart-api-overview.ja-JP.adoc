= API 概要 ({DoughnutChartName})

このセクションでは link:{DoughnutChartLink}.{DoughnutChartName}.html[{DoughnutChartName}]™ に関連する各名前空間を示します。このコントロールでプログラムに使用するいくつかのキー クラスも提供します。このページの名前空間とクラスは当社の API マニュアルに直接リンクします。

[options="header", cols="a,a"]
|====
|クラス|説明

ifdef::win-forms,xaml,android[]
| link:{DoughnutChartLink}.{DoughnutChartName}_members.html[{DoughnutChartName}]
|{DoughnutChartName} コントロールの追加

| link:{DoughnutChartLink}.ringseries_members.html[RingSeries]
| link:{DoughnutChartLink}.{DoughnutChartName}_members.html[{DoughnutChartName}] シリーズ内の 1 つのリングを表します。
endif::win-forms,xaml,android[]

ifdef::wpf,android[]
| link:{DoughnutChartLink}.sliceitem_members.html[SliceItem]
| link:{DoughnutChartLink}.slice_members.html[Slice] エレメントのデータを表します。

| link:{DoughnutChartLink}.slice_members.html[Slice]
|単一のチャート スライスを保持するコンテンツ コントロールを表します。
endif::wpf,android[]
|====
