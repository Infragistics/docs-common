= API の概要

このセクションは、 link:{CategoryChartLink}.{CategoryChartName}.html[{CategoryChartName}]  コントロールに関連する各名前空間をリストします。このコントロールでプログラムする間に使用するいくつかのキー クラスもあります。このページの名前空間とクラスは当社の API マニュアルに直接リンクします。

[options="header", cols="a,a,a"]
|====
|名前空間:|キー クラス:|説明

| link:{CategoryChartLink}_namespace.html[{CategoryChartNamespace}]
|* link:{CategoryChartLink}.{CategoryChartName}.html[{CategoryChartName}]
|カテゴリに固有のプロパティを持つ {ShapeChartXY} から派生されたクラス。
|
|* link:{CategoryChartLink}.{CategoryChartXY}.html[{CategoryChartXY}] 
|X/Y 軸のプロパティを持つ {ShapeChartDomain} から派生されたクラス。
|
|* link:{CategoryChartLink}.{CategoryChartDomain}.html[{CategoryChartDomain}] 
|CategoryChart などのすべてのドメイン チャートの基本クラス。
|
|* link:{CategoryChartLink}.{LegendName}.html[{LegendName}]
|プロットされたデータに関連するコンテキスト情報を持つチャート シリーズのビジュアルを識別するクラスを表します。
|
|* link:{CategoryChartLink}.CategoryChartType.html[CategoryChartType]
|CategoryChart でデータ ビジュアライゼーションのタイプを表します。

|====
