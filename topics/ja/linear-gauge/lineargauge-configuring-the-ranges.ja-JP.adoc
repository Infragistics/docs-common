== 比較範囲の構成

[[_Ref362941461]]

=== 比較範囲構成の概要

{LinearGaugeName} コントロールは、範囲コレクションを作成することで複数の比較範囲をサポートします。

image::images/XamLinearGauge_Overview_11.png[]

各範囲は、スケールのメジャーでの開始値や終了値、塗りつぶし色、および境界線の幅や色を指定することで個別に構成できます。スケール全域での比較範囲のサイズは、内側と外側の余白を調整して構成します。

[[_Ref363721665]]

以下のコードはこの例を実装します。

ifdef::xaml[]

*XAML の場合:*

[source,xaml]
----
<ig:{LinearGaugeName} x:Name="linearGauge" >
      <ig:{LinearGaugeName}.Ranges>
            <ig:{LinearGaugeRange} Brush="Green"
ifdef::sl,wpf,win-universal[]
                                       Caption="range1"
endif::sl,wpf,win-universal[]
                                       StartValue="6"
                                       EndValue="89"
                                       InnerEndExtent="0.5"
                                       InnerStartExtent="0.5"
                                       OuterEndExtent="0.8"
                                       OuterStartExtent="0.8"
                                       Outline="Black"/>
      </ig:{LinearGaugeName}.Ranges>
</ig:{LinearGaugeName}>
----

endif::xaml[]

ifdef::sl[]

*C# の場合:*

[source,csharp]
----
var range1 = new XamLinearGraphRange();
range1.Brush = new SolidColorBrush(Color.FromRgb(0, 255, 0));

range1.StartValue = 6;
range1.EndValue = 89;
range1.InnerEndExtent = .5;
range1.InnerStartExtent = .5;
range1.OuterEndExtent = .8;
range1.OuterStartExtent = .8;
range1.Outline = new SolidColorBrush(Color.FromRgb(0, 0, 0));
----
