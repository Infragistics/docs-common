== マップ キーボード ナビゲーション

[[_Ref320193474]]

=== 概要

{GeoMapName} コントロールがフォーカスを受けると、キーボード ナビゲーションが起動し、マップ コンテンツをキーボード入力でナビゲートできます。

以下の表で、{GeoMapName} コントロールがサポートするキーボード ナビゲーション機能について簡単に説明します。

[options="header", cols="a,a,a"]
|====
|*ユーザーは以下を実行できます...*|*次の使用...*|*説明*

|マップ コンテンツの拡大
|Page Up キー
|キーボードの Page Up キーを押すと、 {GeoMapName}   のマップ コンテンツの中央に拡大係数 0.8 ごとにズームインをします。これは link:{GeoMapLink}.{GeoMapName}~windowscale.html[WindowScale] プロパティの値を減らすことになります。

|マップ コンテンツの縮小
|Page Down キー
|キーボードの Page Down キーを押すと、{GeoMapName} のマップ コンテンツの中央から係数 1.2 ごとにでズームアウトをします。これは link:{GeoMapLink}.{GeoMapName}~windowscale.html[WindowScale]プロパティの値を増やすことになります。

|マップ コンテンツを左（西）にパン
|左矢印キー
|キーボードの左矢印キーを押すと、 {GeoMapName} コントロールのマップ コンテンツを左 (西) にパンします。これは link:{GeoMapLink}.{GeoMapName}~windowpositionhorizontal.html[WindowPositionHorizontal] プロパティの値を減らすことと同じことになります。

|マップ コンテンツを右（東）にパン
|右矢印キー
|キーボードの右矢印キーを押すと、{GeoMapName} コントロールのマップ コンテンツを右 (東) にパンします。これは link:{GeoMapLink}.{GeoMapName}~windowpositionhorizontal.html[WindowPositionHorizontal] プロパティの値を増すことと同じことになります。

|マップ コンテンツを上（北）にパン
|上矢印キー
|キーボードの上矢印キーを押すと、 {GeoMapName} コントロールのマップ コンテンツを上方 (北) にパンします。これは link:{GeoMapLink}.{GeoMapName}~windowpositionvertical.html[WindowPositionVertical] プロパティの値を減らすことと同じことになります。

|マップ コンテンツを下（南）にパン
|下矢印キー
|キーボードの下矢印キーを押すと、{GeoMapName} コントロールのマップ コンテンツを下方 (南) にパンします。これは link:{GeoMapLink}.{GeoMapName}~windowpositionvertical.html[WindowPositionVertical] ロパティの値を増すことと同じことになります。

|マップ コンテンツを表示可能な領域に収まるように拡大
|Home キー
|キーボードの Home キーを押すと、 {GeoMapName} コントロールのマップ コンテンツを表示可能な最大のサイズで表示します。これは  link:{DataChartLinkBase}.seriesviewer~windowrect.html[WindowRect] プロパティをディメンション 0、0、1、1 の {ApiRectClass} に設定することに相当します。
