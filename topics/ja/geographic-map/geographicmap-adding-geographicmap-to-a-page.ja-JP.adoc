== {GeoMapName} をページに追加

[[_Ref320185129]]

=== 概要

{GeoMapName} コントロールは、シェープ ファイルから地理的データ、そして地理的画像マップのビュー モデルから地理的位置を表示するように設計されています。ただし、このトピックは、表示に必要となる最小限のコードとプロパティ設定で {GeoMapName} コントロールを追加することに焦点を当てます。デフォルトで、{GeoMapName} コントロールは、Open Street Map サービスから地理的画像タイルで表示されます。

[[_Ref320186496]]

=== プレビュー

以下は、Open Street Map サービスからの地理的画像タイルがある {GeoMapName} コントロールのプレビューです。

=== 手順

以下では、{GeoMapName} コントロールをページに追加することを手順ごとに説明します。

ifdef::wpf[]

=== インフラジスティックス NuGet パッケージを参照します。

以下の Infragistics NuGet パッケージをアプリケーションに追加します。

** Infragistics.WPF.GeographicMap

NuGet フィードのセットアップと NuGet パッケージの追加の詳細については、link:nuget-feeds.html[NuGet フィード] ドキュメントを参照してください。

endif::wpf[]

ifdef::win-forms[]

=== プロジェクトで Infragistics アセンブリを参照します。

[start=1]
. プロジェクトを右クリックして、[参照の追加] メニュー項目を選択します。
[start=2]
. [参照の追加] ダイアログで、以下のインフラジスティックス アセンブリをブラウズして追加します。
 
* {ApiPlatform}{GeoMapAssembly}{ApiVersion}.dll
* {ApiPlatform}{GeoMapAssemblyBase}{ApiVersion}.dll
* {ApiPlatform}{DataVizAssembly}{ApiVersion}.dll
* {ApiPlatform}Portable.Core{ApiVersion}.dll

endif::win-forms[]


=== アプリケーション ページに名前空間宣言を追加します。

アプリケーション ページで、以下の名前空間宣言を追加します。

ifdef::wpf[]
*XAML の場合:*

[source,xaml]
----
xmlns:ig="http://schemas.infragistics.com/xaml"
----
endif::wpf[]

*Visual Basic の場合:*

[source,vb]
----
Imports {GeoMapNamespace}
----

*C# の場合:*

[source,csharp]
----
using {GeoMapNamespace};
----

=== {GeoMapName} コントロールをページに追加します。

以下のコードは、表示に必要となる最小限のコードとプロパティ設定で {GeoMapName} コントロールを追加することを示します。

ifdef::wpf[]
*XAML の場合:*

[source,xaml]
----
<Grid x:Name="LayoutRoot">
    <ig:XamGeographicMap x:Name="GeoMap" />
</Grid>
----
endif::wpf[]

*Visual Basic の場合:*

[source,vb]
----
Dim geoMap As New {GeoMapName}()
ifdef::wpf[]
Me.LayoutRoot.Children.Add(geoMap)
endif::wpf[]
ifdef::win-forms[]
Me.Controls.Add(geoMap)
endif::win-forms[]
----

*C# の場合:*

[source,csharp]
----
var geoMap = new {GeoMapName}();
ifdef::wpf[]
this.LayoutRoot.Children.Add(geoMap);
endif::wpf[]
ifdef::win-forms[]
this.Controls.Add(geoMap)
endif::win-forms[]
----

=== 結果を確認します。

結果を確認するために、プロジェクトをビルドおよび実行します。手順を正しく実装した場合、{GeoMapName}   は上記の<<_Ref320186496,プレビュー>> セクションで示したように表示されるはずです。
