= マーカー値の表示

このトピックは、コード例を使って、マーカー値を link:{DataChartLink}.{DataChartName}.html[{DataChartName}] コントロールで表示する方法を説明します。

== 概要

{DataChartName} コントロールで、データ ポイントの値をマーカーに表示できます。(図 1) これを行うには、 link:{DataChartLink}.markerseries.html[MarkerSeries] クラスから継承した link:datachart-series.html[チャート シリーズ] の link:{DataChartLink}.markerseries{ApiProp}markertemplate.html[MarkerTemplate] プロパティに、カスタム マーカー データ テンプレートを追加します。

== コード例

以下のコードは、 link:{DataChartLink}.columnseries.html[ColumnSeries] のマーカー内に値を表示する方法を示しています。

.注:
[NOTE]
====
示されている方法は、 link:{DataChartLink}.markerseries.html[MarkerSeries] ベース クラスを継承する他のシリーズにも使用できます。
====

ifdef::wpf,win-universal[]

*XAML の場合:*

----
<ig:{DataChartName} x:Name="dataChart" >
    <ig:{DataChartName}.Series>
        <ig:ColumnSeries ItemsSource="{Binding}"
                         ValueMemberPath="Value" 
                         XAxis="{Binding ElementName=xAxis}"
                         YAxis="{Binding ElementName=yAxis}">
             <ig:ColumnSeries.MarkerTemplate>
                  <DataTemplate>
                      <Grid Width="40" Height="30" Margin="0,0,0,0">
                           <Border CornerRadius="5" Background="#72010101" 
                                   BorderBrush="DimGray" BorderThickness="0.5">
                                   <!--注: TextBlock is のテキスト プロパティは、
                                   データ ソースを構成するアイテムのプロパティとバインドします -->
                              <TextBlock Margin="2" Text="{Binding Item.Value}" 
                                         Foreground="White" FontSize="15" 
                                         HorizontalAlignment="Center" />
                           </Border>
                      </Grid>
                  </DataTemplate>
             </ig:ColumnSeries.MarkerTemplate>
        </ig:ColumnSeries >
    </ig:{DataChartName}.Series>
</ig:{DataChartName}>
----

endif::wpf,win-universal[]
