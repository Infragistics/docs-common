=== チャート要件

チャート コントロールは シリーズと呼ばれ、さまざまなデータ視覚化をサポートします。このシリーズ オブジェクトは広範なデータ ソースを可視化できます。チャート コントロールにサポートされるシリーズの一覧は、 link:datachart-series-types.html[シリーズ タイプ] トピックを参照してください。各シリーズ タイプは一定の要件を満たす (データ列の数およびタイプなど) データおよびデータをサポートされる軸によって、そのデータがプロットできます。各シリーズの要件については、 link:datachart-series-requirements.html[シリーズ要件]トピックを参照してください。

このトピックは link:{DataChartLink}.areaseries.html[AreaSeries] 1 つと 2 つの軸 (CategoryXAxis および NumericYAxis)、そして link:resources-sample-energy-data.html[サンプル エネルギー データ]を使用します。

[[_Ref396236025]]

=== アプリケーションの作成

[start=1]
. {PlatformIDE} またはそれ以降のバージョンで {PlatformName} アプリケーション プロジェクトを作成し、名前を *SampleApp* に設定します。

[start=2]
. 以下の名前空間をメイン コード ファイルにインポートします。

ifdef::xaml[]

*C# の場合:*

ifdef::wpf,win-universal[]
----
using {DataChartNamespace};
 
----
endif::wpf,win-universal[]

ifdef::xamarin[]
---- 
using Infragistics.XamarinForms.Controls.Charts; 
----
endif::xamarin[]

endif::xaml[]

ifdef::wpf,win-universal[]

*Visual Basic の場合:*

----
Imports {DataChartNamespace}
 
----
endif::wpf,win-universal[]

ifdef::android[]

*Java の場合:*

[source,js]
----
import com.infragistics.controls.DataChartView;
import com.infragistics.controls.CategoryXAxis;
import com.infragistics.controls.NumericYAxis;
import com.infragistics.controls.AreaSeries;
----

endif::android[]

=== データ ソースの作成

[start=1]
. サンプル データ コードを link:resources-sample-energy-data.html[サンプル エネルギー データ] リソースからコピーして *SampleApp* プロジェクトに追加します。

[start=2]
. カテゴリ サンプル データのインスタンスを作成します。

ifdef::xaml[]

*C# の場合:*

[source,csharp]
----
var data = new EnergyProductionDataSample();
----

endif::xaml[]

ifdef::wpf,win-universal[]

*Visual Basic の場合:*

[source,vb]
----
Dim data As New EnergyProductionDataSample()
----

endif::wpf,win-universal[]

ifdef::android[]

*Java の場合:*

[source,js]
----
EnergyProductionDataSample data = new EnergyProductionDataSample();
----

endif::android[]

[[_Ref396236032]]

=== 軸の作成

横軸でラベルを表示するためのカテゴリ X 軸を作成します

ifdef::xaml[]

*C# の場合:*

ifdef::wpf,win-universal[]
----
var xAxis = new CategoryXAxis();
xAxis.ItemsSource = data;
xAxis.Label = "{}{County}"; 
----
endif::wpf,win-universal[]

ifdef::xamarin[]
----
var xAxis = new CategoryXAxis();
xAxis.ItemsSource = data;
 
xAxis.Label = "County";
----
endif::xamarin[]

endif::xaml[]

ifdef::wpf,win-universal[]

*Visual Basic の場合:*

----
Dim xAxis = New CategoryXAxis() 
xAxis.ItemsSource = data
xAxis.Label = "{}{County}" 
----

endif::wpf,win-universal[]

ifdef::android[]

*Java の場合:*

[source,js]
----
CategoryXAxis xAxis = new CategoryXAxis();
xAxis.setDataSource(data);
xAxis.setLabel("Country");
----

endif::android[]

縦軸で値を表示するための数値 Y 軸を作成します

ifdef::xaml[]

*C# の場合:*

[source,csharp]
----
var yAxis = new NumericYAxis();
----

endif::xaml[]

ifdef::wpf,win-universal[]

*Visual Basic の場合:*

[source,vb]
----
Dim yAxis = New NumericYAxis()
----

endif::wpf,win-universal[]

ifdef::android[]

*Java の場合:*

[source,js]
----
NumericYAxis yAxis = new NumericYAxis();
----

endif::android[]

[[_Ref396236034]]

=== シリーズの作成

サンプル データ ソースへのバインディングおよびデータ マッピングが設定されるシリーズのインスタンスを作成します。また、シリーズに前のセクションで作成した 2 つの軸の設定が必要です。

ifdef::xaml[]

*C# の場合:*

[source,csharp]
----
var series = new AreaSeries();
series.ValueMemberPath = "Coal";
series.ItemsSource = data;
series.XAxis = xAxis;
series.YAxis = yAxis;
----

endif::xaml[]

ifdef::wpf,win-universal[]

*Visual Basic の場合:*

[source,vb]
----
Dim series = New AreaSeries() 
series.ValueMemberPath = "Coal"
series.ItemsSource = data
series.XAxis = xAxis
series.YAxis = yAxis
----

endif::wpf,win-universal[]

ifdef::android[]

*Java の場合:*

[source,js]
----
AreaSeries series = new AreaSeries();
series.setValueMemberPath("Coal");
series.setDataSource(data);
series.setXAxis(xAxis);
series.setYAxis(yAxis);
----

endif::android[]

[[_Ref396236037]]

=== チャートの作成

1 つのシリーズおよび 2 つの軸の設定を持つチャート コントロールのインスタンスを作成します。

ifdef::xaml[]

*C# の場合:*

[source,csharp]
----
var chart = new {DataChartName}();
chart.Axes.Add(xAxis);
chart.Axes.Add(yAxis);
chart.Series.Add(series);
----

endif::xaml[]

ifdef::wpf,win-universal[]

*Visual Basic の場合:*

[source,vb]
----
Dim chart As New {DataChartName}() 
chart.Axes.Add(xAxis)
chart.Axes.Add(yAxis)
chart.Series.Add(series)
----

endif::wpf,win-universal[]

ifdef::android[]

*Java の場合:*

[source,js]
----
DataChartView chart = new DataChartView(root.getContext());
chart.addAxis(xAxis);
chart.addAxis(yAxis);
chart.addSeries(series);
----

endif::android[]
