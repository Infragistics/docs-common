== タッチ ジェスチャ

[[_Ref320185129]]

=== 概要

タッチ ジェスチャは、 _{DataChartName}_   コントロールでユーザーがコンテンツをナビゲートおよび操作することを可能にするスクリーンの操作です。

ifdef::wpf,win-universal[]
.注:
[NOTE]
====
_{DataChartName}_   コントロールのデフォルト モードであるパン アンド ズーム動作は、ズームのためにパンとマウスのためにタブ アンド ドラッグ ジェスチャを使用します。link:{DataChartLinkBase}.{DataChartBase}{ApiProp}defaultinteraction.html[DefaultInteraction] プロパティは設定されていません。ただし、このプロパティが明示的に DragZoom に設定されている場合、タッチ ジェスチャはチャート コンテンツのズームを強制します。
====
endif::wpf,win-universal[]

[[_Ref335405092]]

=== タッチ ジェスチャの概要

以下の表に、 _{DataChartName}_   コントロールの主な機能を簡単に説明します。

[options="header", cols="a,a,a"]
|====
|目的|方法|詳細

|チャート コンテンツの拡大
|拡大ジェスチャ
|2 本の指で面をタッチし、互いを遠ざけるようにドラッグします。

|チャート コンテンツの縮小
|縮小ジェスチャ
|2 本の指で面をタッチし、互いを近づけるようにドラッグします。

|チャート コンテンツのパン
|パン ジェスチャ
|面をタッチして、次にタッチして直ちに指を離さずにパンのための希望の方向に指を移動します。

|チャートですべてのシリーズのヒントを表示
|タブ アンド ドラッグ ジェスチャ
|数秒間面をタッチしてそのままにしておき、指をドラッグします。

|====
