== ナビゲーション操作

{DataChartName} コントロールでは、表 3 に示されるように、ユーザーは様々な方法でパンニングとズームを実行できます。

[options="header", cols="a,a,a"]
|====
|作業|キーボードを使用|マウスを使用

|パン
|矢印キー
|Shift キー + マウス ドラッグ

|ズームイン
|Page Up キー
|マウス ホイールをスクロール ダウン

|ズームアウト
|Page Down キー
|マウス ホイールをスクロール アップ

|チャート プロット領域に合わせる
|Home キー
|なし

|領域ズーム
|なし
|チャート プロット領域内ドラッグすると、ズームインするチャートの領域を選択できます。
