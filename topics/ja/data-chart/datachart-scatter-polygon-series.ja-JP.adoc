== 散布多角形シリーズ

[[_Ref320193474]]

=== 概要

{DataChartName} コントロールで、link:{DataChartLink}.ScatterPolygonSeries_members.html[ScatterPolygonSeries] は多角形を使用してデータを表示するビジュアル要素です。このシリーズのタイプは任意の図形を描画できます。link:{DataChartLink}.ScatterPolygonSeries_members.html[ScatterPolygonSeries] は、データがポリラインの代わりに多角形で描画されることを除いて、link:{DataChartLink}.ScatterPolylineSeries_members.html[ScatterPolylineSeries] とほどんど同様に機能します。

[[_Ref320651783]]

=== データ要件

{DataChartName} コントロールのシリーズの他のタイプと同様、link:{DataChartLink}.ScatterPolygonSeries_members.html[ScatterPolygonSeries] には、データ バインディングのための link:{DataChartLink}.series~{ApiDataSource}.html[{ApiDataSource}] プロパティがあります。このプロパティは、link:http://msdn.microsoft.com/ja-jp/library/system.collections.ienumerable.aspx[IEnumerable] インターフェイスを実装するオブジェクトにバインドできます。さらに、このオブジェクトの各項目には、link:http://msdn.microsoft.com/ja-jp/library/system.collections.ienumerable.aspx[IEnumerable]<link:http://msdn.microsoft.com/ja-jp/library/system.collections.ienumerable.aspx[IEnumerable]<pick:[wpf="link:http://msdn.microsoft.com/ja-jp/library/system.windows.point.aspx[Point]"]pick:[win-forms"{GeoMapNamespace}.Point"]>> 構造を使用して図形のポイント位置 (X と Y 値) を保存するデータ列が 1 つ必要です。このデータ列は、link:{DataChartLink}.ShapeSeriesBase~ShapeMemberPath.html[ShapeMemberPath] プロパティにマップされます。link:{DataChartLink}.ScatterPolygonSeries_members.html[ScatterPolygonSeries] は、{DataChartName} コントロールで多角形をプロットするために、このマップされたデータ列のポイントを使用します。

[[_Ref320651790]]

=== 例

以上のスクリーンショットでのデータの構造は以下です。

*C# の場合:*
[source,csharp]
----
public class Node
{
    // A list of polygons and each polygon has a list of points.
    public List<List<Point>> Points { get; set; }
}
----

*Visual Basic の場合:*
[source,vb]
----
Public Class Node
    ' A list of polygons and each polygon has a list of points.
    Private _points As List(Of List(Of Point))
    public Property List(Of List(Of Point)) Points
        Get
            Return _points
        End Get
        Set
            _points = value
        End Set
    End Property
End Class
----

以下のコードは link:{DataChartLink}.ScatterPolygonSeries_members.html[ScatterPolygonSeries] をデータにバインドします。

ifdef::wpf[]
*XAML の場合:*

[source,xaml]
----
<ig:{DataChartName} x:Name="dataChart">
    <ig:{DataChartName}.Axes>
        <ig:NumericXAxis x:Name="xAxis"/>
        <ig:NumericYAxis x:Name="yAxis"/>
    </ig:{DataChartName}.Axes>
    <ig:{DataChartName}.Series>
        <ig:ScatterPolygonSeries ItemsSource="{Binding Path=Nodes}"
                                 ShapeMemberPath="Points"
                                 XAxis="{Binding ElementName=xAxis}"
                                 YAxis="{Binding ElementName=yAxis}"/>
    </ig:{DataChartName}.Series>
</ig:{DataChartName}>
----
endif::wpf[]

*Visual Basic の場合:*

[source,vb]
----
' ScatterPolygonSeries requires numeric X and Y axis
Dim xAxis = New NumericXAxis()
Dim yAxis = New NumericYAxis()
Me.dataChart.Axes.Add(xAxis)
Me.dataChart.Axes.Add(yAxis)

' create and set data binding to the ScatterPolygonSeries
Dim polygonSeries = New ScatterPolygonSeries()
polygonSeries.{ApiDataSource} = Nodes
polygonSeries.ShapeMemberPath = "Points"
polygonSeries.XAxis = xAxis
polygonSeries.YAxis = yAxis

' add the ScatterPolygonSeries to the the {DataChartName}
Me.dataChart.Series.Add(polygonSeries)
----

*C# の場合:*

[source,csharp]
----
// ScatterPolygonSeries requires numeric X and Y axis
var xAxis = new NumericXAxis();
var yAxis = new NumericYAxis();
this.dataChart.Axes.Add(xAxis);
this.dataChart.Axes.Add(yAxis);

// create and set data binding to the ScatterPolygonSeries
var polygonSeries = new ScatterPolygonSeries();
polygonSeries.{ApiDataSource} = Nodes;
polygonSeries.ShapeMemberPath = "Points";
polygonSeries.XAxis = xAxis;
polygonSeries.YAxis = yAxis;

// add the ScatterPolygonSeries to the the {DataChartName}
this.dataChart.Series.Add(polygonSeries);
----
