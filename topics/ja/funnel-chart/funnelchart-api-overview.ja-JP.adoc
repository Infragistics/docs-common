= API の概要

このセクションは、 link:{FunnelChartLink}.{FunnelChartName}.html[{FunnelChartName}] コントロールに関連する各名前空間をリストします。このコントロールでプログラムする間に使用するいくつかのキー クラスもあります。このページの名前空間とクラスはインフラジスティックス API マニュアルに直接リンクします。

|====
|名前空間：|キー クラス:
