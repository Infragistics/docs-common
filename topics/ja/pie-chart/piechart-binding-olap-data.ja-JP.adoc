== OLAP データを含む {PieChartName} の使用

[[_Ref321135670]]

=== OLAP Visuals

{PieChartName} を使用した OLAP データの可視化は、 link:{ApiPlatform}controls.charts.olap{ApiVersion}~{PieChartNamespace}.olapaxis_members.html[OlapAxis] と呼ばれる新しい軸タイプによって行います。

OlapAxis は link:{ApiPlatform}controls.charts.olap{ApiVersion}~{PieChartNamespace}.olapaxis~datasource.html[DataSource] および link:{ApiPlatform}controls.charts.olap{ApiVersion}~{PieChartNamespace}.olapaxis~olapaxissource.html[OlapAxisSource] の 2 つの主要なプロパティで構成されています。

DataSoure はデータ バインディングに使用し、OlapAxisSource はデータ描画の際の軸タイプを決定する場合に使用します。円チャートで OLAP データを視覚化するには、これらのプロパティを両方とも設定することが重要です。

軸タイプは Columns または Rows、つまりデータ ソースからの列または行になります。

データ ソースの各列は個別の円チャートとして描画され、円チャートの行は以下の図に示すようにスライスとして表されます。

image::images/PieChart_Binding_OLAP_Data_01.png[]

{PieChartName} コントロールは、複数の列を含むデータを複数の円チャートとして描画します。円チャートはデータ ソースから列ごとに 1 つあります。前述の例のように、列の行は、対応する円でスライスとして描画されます。この動作は以下のスクリーン ショットで示しています。

image::images/PieChart_Binding_OLAP_Data_02.png[]

デフォルトでは、円チャートは 1 方向の列と 1 方向の行を表示し、1 つの円チャートになります。列と行は展開されません。

詳細については、 link:piechart-user-interactions-with-olap-data.html[OLAP データを視覚化するための {PieChartName} でのユーザー操作]を参照してください。

=== コード例

このコーディング例は、OLAP データの表示に最低限必要な {PieChartName} コントロールのプロパティ設定を示します。

ifdef::xaml[]

*XAML の場合:*

[source,xaml]
----
<ig:XamOlapPieChart 
    DataSource="{Binding DataSource}" 
    OlapAxisSource="Columns" />
----

endif::xaml[]

ifdef::xaml[]

*C# の場合:*

[source,csharp]
----
XamOlapPieChart chart = new XamOlapPieChart();
chart.DataSource = new MainViewModel().DataSource;
chart.OlapAxisSource = OlapAxisSource.Columns;
----

endif::xaml[]

ifdef::win-forms[]

*C# の場合:*

[source,csharp]
----
XamOlapPieChart chart = new XamOlapPieChart();
chart.DataSource = new MainViewModel().DataSource;
chart.OlapAxisSource = OlapAxisSource.Columns;
----

endif::win-forms[]

ifdef::xaml[]

*Visual Basic の場合:*

[source,vb]
----
Dim chart As New XamOlapPieChart()
chart.DataSource = New MainViewModel().DataSource
chart.OlapAxisSource = OlapAxisSource.Columns
----

endif::xaml[]

ifdef::win-forms[]

*Visual Basic の場合:*

[source,vb]
----
Dim chart As New XamOlapPieChart()
chart.DataSource = New MainViewModel().DataSource
chart.OlapAxisSource = OlapAxisSource.Columns
----

endif::win-forms[]
