== 比較マーカーの構成

[[_Ref363745014]]

=== 比較マーカー構成の概要

比較マーカーは、パフォーマンス バーが示す値と比較された値をスケールに表示します。比較値は、目標売上高などの目標値または値の超過限度を示す制限値 (体温計の37℃の発熱の目安) にすることができます。

比較目盛マーカーが示す値およびスケール上の位置は、`TargetValue` プロパティで設定します。比較マーカーは、そのスケール全域で位置やサイズ ( link:{BulletGraphLink}.{BulletGraphName}{ApiProp}targetvalueinnerextent.html[TargetValueInnerExtent] および link:{BulletGraphLink}.{BulletGraphName}{ApiProp}targetvalueouterextent.html[TargetValueOuterExtent])、幅 ( link:{BulletGraphLink}.{BulletGraphName}{ApiProp}targetvaluebreadth.html[TargetValueBreadth]) およびルック アンド フィール (塗りつぶし色 - link:{BulletGraphLink}.{BulletGraphName}{ApiProp}targetvaluebrush.html[TargetValueBrush]、境界線の線幅 - link:{BulletGraphLink}.{BulletGraphName}{ApiProp}targetvaluestrokethickness.html[TargetValueStrokeThickness]、境界線の色 – link:{BulletGraphLink}.{BulletGraphName}{ApiProp}targetvalueoutline.html[TargetValueOutline]) に関して各プロパティでカスタマイズできます。

=== 例

以下のスクリーンショットは、下記設定の結果、 _{BulletGraphName}_   の外観がどのようになるかを示しています。

image::images/BulletGraph_Configuring_the_Comparative_Marker_2.png[]

以下のコードはこの例を実装します。

ifdef::xaml[]

*XAML の場合:*

[source,xaml]
----
<ig:{BulletGraphName} x:Name="bulletGraph"
                       TargetValue="70"
                       TargetValueBreadth="10"
                       TargetValueBrush="Yellow"
                       TargetValueOutline="#ff006400"
                       TargetValueStrokeThickness="2"
                       TargetValueInnerExtent="0.3"
                       TargetValueOuterExtent="0.8" />
----

endif::xaml[]

ifdef::xaml[]

*C# の場合:*

[source,csharp]
----
bulletGraph.TargetValue = 70;
bulletGraph.TargetValueBreadth = 10;
bulletGraph.TargetValueBrush = new SolidColorBrush(Color.FromRgb(0, 64, 0));
bulletGraph.TargetValueOutline = new SolidColorBrush(Color.Yellow);
bulletGraph.TargetValueStrokeThickness = 2;
bulletGraph.TargetValueInnerExtent = .3;
bulletGraph.TargetValueOuterExtent = .8;
----

endif::xaml[]

ifdef::win-forms[]

*C# の場合:*

[source,csharp]
----
bulletGraph.TargetValue = 70;
bulletGraph.TargetValueBreadth = 10;
bulletGraph.TargetValueBrush = new SolidColorBrush(Color.Yellow);
bulletGraph.TargetValueOutline = new SolidColorBrush(Color.FromArgb(0, 64, 0));
bulletGraph.TargetValueStrokeThickness = 2;
bulletGraph.TargetValueInnerExtent = .3;
bulletGraph.TargetValueOuterExtent = .8;
----

endif::win-forms[]

ifdef::wpf,sl[]

*VB の場合:*

[source,vb]
----
bulletGraph.TargetValue = 70
bulletGraph.TargetValueBreadth = 10
bulletGraph.TargetValueBrush = New SolidColorBrush(Color.FromRgb(0, 64, 0))
bulletGraph.TargetValueOutline = New SolidColorBrush(Color.Yellow)
bulletGraph.TargetValueStrokeThickness = 2
bulletGraph.TargetValueInnerExtent = .3
bulletGraph.TargetValueOuterExtent = .8
----

endif::wpf,sl[]

ifdef::win-forms[]

*VB の場合:*

[source,vb]
----
bulletGraph.TargetValue = 70
bulletGraph.TargetValueBreadth = 10
bulletGraph.TargetValueBrush = New SolidColorBrush(Color.FromArgb(0, 64, 0))
bulletGraph.TargetValueOutline = New SolidColorBrush(Color.Yellow)
bulletGraph.TargetValueStrokeThickness = 2
bulletGraph.TargetValueInnerExtent = .3
bulletGraph.TargetValueOuterExtent = .8
----

endif::win-forms[]

ifdef::android[]

*Java の場合:*

[source,js]
----
bulletGraph.setTargetValue(70);
bulletGraph.setTargetValueBreadth(10);
bulletGraph.setTargetValueBrush(new SolidColorBrush(Color.YELLOW););
bulletGraph.setTargetValueOutline(new SolidColorBrush(Color.parseColor("#006400"));
bulletGraph.setTargetValueStrokeThickness(2);
bulletGraph.setTargetValueInnerExtent(.3);
bulletGraph.setTargetValueOuterExtent(.8);
----

endif::android[]
