== パフォーマンス バーの構成

[[_Ref362941119]]

=== パフォーマンス バーの外観構成の概要

パフォーマンス バーはスケール範囲の先頭から開始しなければなりません。終了位置のみ構成可能です。終了位置でバーの長さを効果的に構成できます。終了位置は link:{BulletGraphLink}.{BulletGraphName}{ApiProp}value.html[Value] プロパティで処理されます。

スケール全域のディメンションでの位置は、 link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valueinnerextent.html[ValueInnerExtent] および link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valueouterextent.html[ValueOuterExtent] プロパティにより、グラフ領域の端に対して構成できます。パフォーマンス バーの位置を構成すると幅も定義されます。

image::images/BulletGraph_Configuring_the_Performance_Bar_1.png[]

バーのルック アンド フィールは、各プロパティ ( link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valuebrush.html[ValueBrush]、 link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valueoutline.html[ValueOutline] および link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valuestrokethickness.html[ValueStrokeThickness]) を使用して塗りつぶし色、境界線の色および境界線の線幅をカスタマイズできます。

=== 例

以下のスクリーンショットは、下記設定の結果、 _{BulletGraphName}_   の外観がどのようになるかを示しています。

[options="header", cols="a,a"]
|====
|プロパティ|値

| link:{BulletGraphLink}.{BulletGraphName}{ApiProp}value.html[Value]
|"85"

| link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valuebrush.html[ValueBrush]
|"#ffffa500"

| link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valueoutline.html[ValueOutline]
|"#ffa52a2a"

| link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valueinnerextent.html[ValueInnerExtent]
|"0.3"

| link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valueouterextent.html[ValueOuterExtent]
|"0.8"

| link:{BulletGraphLink}.{BulletGraphName}{ApiProp}valuestrokethickness.html[ValueStrokeThickness]
|"3"

|====

image::images/BulletGraph_Configuring_the_Performance_Bar_2.png[]

以下のコードはこの例を実装します。

ifdef::xaml[]

*XAML の場合:*

[source,xaml]
----
<ig:{BulletGraphName} Value="85"
                 ValueBrush="#ffffa500"
                 ValueInnerExtent="0.3"
                 ValueOuterExtent="0.8"
                 ValueOutline="#ffa52a2a"
                 ValueStrokeThickness="3" />
----

endif::xaml[]

ifdef::xaml[]

*C# の場合:*

[source,csharp]
----
bulletGraph.Value = 85;
bulletGraph.ValueBrush = new SolidColorBrush(Color.FromRgb(255, 165, 0));
bulletGraph.ValueInnerExtent = .3;
bulletGraph.ValueOuterExtent = .8;
bulletGraph.ValueOutline = new SolidColorBrush(Color.FromRgb(165,42,42));
bulletGraph.ValueStrokeThickness = 3;
----

endif::xaml[]

ifdef::win-forms[]

*C# の場合:*

[source,csharp]
----
bulletGraph.Value = 85;
bulletGraph.ValueBrush = new SolidColorBrush(Color.FromArgb(255, 165, 0));
bulletGraph.ValueInnerExtent = .3;
bulletGraph.ValueOuterExtent = .8;
bulletGraph.ValueOutline = new SolidColorBrush(Color.FromArgb(165, 42, 42));
bulletGraph.ValueStrokeThickness = 3;
----
endif::win-forms[]

ifdef::wpf,sl[]

*VB の場合:*

[source,vb]
----
bulletGraph.Value = 85
bulletGraph.ValueBrush = New SolidColorBrush(Color.FromRgb(255, 165, 0))
bulletGraph.ValueInnerExtent = .3
bulletGraph.ValueOuterExtent = .8
bulletGraph.ValueOutline = New SolidColorBrush(Color.FromRgb(165, 42, 42))
bulletGraph.ValueStrokeThickness = 3
----

endif::wpf,sl[]

ifdef::win-forms[]

*VB の場合:*

[source,vb]
----
bulletGraph.Value = 85
bulletGraph.ValueBrush = New SolidColorBrush(Color.FromArgb(255, 165, 0))
bulletGraph.ValueInnerExtent = .3
bulletGraph.ValueOuterExtent = .8
bulletGraph.ValueOutline = New SolidColorBrush(Color.FromArgb(165, 42, 42))
bulletGraph.ValueStrokeThickness = 3
----

endif::win-forms[]

ifdef::android[]

*Java の場合:*

[source,js]
----
bulletGraph.setValue(85);
bulletGraph.setValueBrush(new SolidColorBrush(Color.parseColor("#FF8000")));
bulletGraph.setValueInnerExtent(.3);
bulletGraph.setValueOuterExtent(.8);
bulletGraph.setValueOutline(new SolidColorBrush(Color.parseColor("#A52A2A")));
bulletGraph.setValueStrokeThickness(3);
----

endif::android[]
