= Supported Column Types

This topic introduces supported column types in the link:{DataGridLink}.{DataGridName}.html[{DataGridName}] control.

For a detailed overview of working with Columns, please see the link:datagrid-working-with-columns.html[Working with Columns] topic.

==== Supported Column types:

* link:{DataGridLink}.TextColumn_members.html[TextColumn] is used for displaying formatted text in associated cells; this is the default column type used to display data of the string data type.
* link:{DataGridLink}.NumericColumn_members.html[NumericColumn] is used for displaying a formatted numeric value in a cell and enables the control of decimal place placement within cells, setting minimum and maximum value thresholds and displaying fractional digits.
* link:{DataGridLink}.DateTimeColumn_members.html[DateTimeColumn] is used for displaying pick:[android="`Calendar`"]pick:[xaml="`DateTime`"] objects in a cell and provides predefined DateTime formats including: `DateTimeLong`, `DateTimeShort`, `DateLong`, `DateShort`, `TimeLong`, and `TimeShort`.
* link:{DataGridLink}.ImageColumn_members.html[ImageColumn] is used for displaying an image within a cell and exposes options for image stretch customization for cells.
* link:{DataGridLink}.TemplateColumn_members.html[TemplateColumn] is used in conjunction with a cell template, allowing you to apply a custom template to the columns cell placement.
