= Getting Started with {DataPieChartName}

== Getting Started

To get started using the link:{DataPieChartLink}.{DataPieChartName}.html[{DataPieChartName}] control, see the link:data-pie-chart-data-binding.html[Data Binding] topic.

== Essential {DataPieChartName} Properties

The following is a summary of some of the more essential properties of the Pie Chart control.

ifdef::wpf[] 

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}itemssource.html[ItemsSource] - Stores a reference to the Pie Chart's data.

endif::wpf[]

ifdef::win-forms[]

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}datasource.html[DataSource] - Stores a reference to the Pie Chart’s data.

endif::win-forms[]

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}includedproperties.html[IncludedProperties] - A string array specifying the property names on the underlying data model to be used as the value and label. This can be used instead of `ValueMemberPath`, listed below.

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}excludedproperties.html[ExcludedProperties] - A string array specifying the property names on the underlying data model to be ignored.

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}valuememberpath.html[ValueMemberPath] - A string specifying the property name on the data model to be used as value, i.e. the arc length of each pie slice.

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}labelmemberpath.html[LabelMemberPath] - This string specifies the property name on the data model to be used as label, i.e. the text that appears on each pie slice and inside the legend.

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}legend.html[Legend] - Stores an optional reference to the ItemLegend object to be associated with the chart.

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}legendslicelabelcontentmode.html[LegendSliceLabelContentMode] - An enumerated type that determines the content of the label for the Legend of the slices plotted in the chart.

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlightingbehavior.html[HighlightingBehavior] - An enumerated type that determines how a slice or series in the chart is highlighted.

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlightingmode.html[HighlightingMode] - An enumerated type that determines the appearance of highlighted slices in the chart.

* pick:[wpf="link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlighteditemssource.html[HighlightedItemsSource]"] pick:[win-forms="link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlighteddatasource.html[HighlightedDataSource]"] - A collection representing a subset of the data plotted in the chart, used for a highlighting overlay to visualize that subset of data.

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}highlightedvaluesdisplaymode.html[HighlightedValuesDisplayMode] - An enumerated property that allows you to enable an overlay for representing the `HighlightedDataSource`.

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}radiusextent.html[RadiusExtent] Determines the radius of the pie chart. When set to 1, the radius is equal to the maximum of the control’s Width and Height. The valid value range is between 0 and 1.

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}selectionbehavior.html[SelectionBehavior] - An enumerated type that determines whether single, multiple, or no selection is enabled in the chart.

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}selectionbehavior.html[SelectionMode] - An enumerated type that determines the behavior of the slices when they are selected.

* link:{DataPieChartLink}.{DomainChartName}{ApiProp}selectedseriesitems.html[SelectedSeriesItems] - A collection of the underlying data items for each slice that describes which pie slices are currently selected.

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}slicelabelcontentmode.html[SliceLabelContentMode] - An enumerated type that specifies what to show in the labels for each slice.

* link:{DataPieChartLink}.{DataPieChartBase}{ApiProp}slicelabelpositionmode.html[SliceLabelPositionMode] - An enumerated type that specifies how labels are arranged.

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}startangle.html[StartAngle] - The offset angle, in degrees, at which pie slices start to be rendered.

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}sweepdirection.html[SweepDirection] - Determines the angular direction in which pie slices are rendered, beginning from the StartAngle.

* link:{DataPieChartLink}.{PieChartBase}{ApiProp}brushes.html[Brushes] - An optional reference to a collection of brushes, used in sequential order to color the pie slices.
