Sorting is done by setting a sorting condition on a worksheet level object on either column or row.You can sort columns or rows in ascending or descending order.

The sort conditions in a sheet are only reapplied when sort conditions are added, removed, modified, or when the link:{ExcelEngineApiLink}infragistics.documents.excel.sorting.worksheetsortsettings~reapplysortconditions.html[ReapplySortConditions] method is called on the sheet.Columns or rows will be sorted within the region. 'Rows' is the default sort type.

The following sort settings are available to set on rows:

[options="header", cols="a,a"]
|====
|Sort condition type|Description

| link:{ExcelEngineApiLink}infragistics.documents.excel.sorting.worksheetsortsettings~sorttype.html[SortType] 
|In worksheet Column/Row to be sorted based on their condition.

|link:{ExcelEngineApiLink}infragistics.documents.excel.sorting.sortsettings%601~casesensitive.html[CaseSensitive]
|True or false depends whether strings should be compared case-sensitively when they are sorted.

|link:{ExcelEngineApiLink}infragistics.documents.excel.sorting.worksheetsortsettings~setregion.html[SetRegion]
|Set the region to apply the sort setting.

|link:{ExcelEngineApiLink}infragistics.documents.excel.sorting.sortsettings%601~sortconditions.html[SortConditions]
|Sort conditions to use when sorting the region of data.

|====

== Code Example:

The following code snippet shows how to perform Worksheet level sorting on the {ExcelEngineName}.Set the region to apply sorting from column B8 to column B15.

This example uses a code snippet for programmatically creating a workbook

Following is the code that implements this example.

*In C#:*

[source,csharp]
----
Worksheet Worksheet1 = Workbook.Worksheets[0];
Worksheet1.SortSettings.SortType = Infragistics.Documents.Excel.WorksheetSortType.Rows;
Worksheet1.SortSettings.CaseSensitive = true;
Worksheet1.SortSettings.SetRegion("B8:B15");
Worksheet1.SortSettings.SortConditions.Add(new RelativeIndex(0), new OrderedSortCondition(SortDirection.Ascending);
----

*In Visual Basic:*

[source,vb]
----
Worksheet Worksheet1 = Workbook.Worksheets(0)
Worksheet1.SortSettings.SortType = Infragistics.Documents.Excel.WorksheetSortType.Rows
Worksheet1.SortSettings.CaseSensitive = True
Worksheet1.SortSettings.SetRegion("B8:B15")
Worksheet1.SortSettings.SortConditions.Add(New RelativeIndex(0), New OrderedSortCondition(SortDirection.Ascending)
----
