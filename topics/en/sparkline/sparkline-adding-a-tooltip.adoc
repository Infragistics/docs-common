== Adding a Tooltip

[[_Ref317179380]]

=== Overview

The tooltip is added by setting the link:{SparklineLink}.{SparklineName}~tooltipvisibility.html[ToolTipVisibility] property. If this property is set to visible, the tooltip displays the following default values:

* The first data point value.
* The last data point value.
* The highest data point value.
* The lowest data point value.

image::images/Sparkline_Visual_Elements_7.png[]

By default (if no customization settings have been applied) the tooltip displays as the four values listed above, first, last, highest and lowest data points.
ifdef::sl,wpf,win-phone,win-universal,xamarin[]
For details about configuring  _{SparklineName}_   tooltips, refer to the link:sparkline-configuring-the-tooltip.html[Configuring the Tooltip] topic.
endif::sl,wpf,win-phone,win-universal,xamarin[]

=== Code
ifdef::sl,wpf,win-phone,win-universal[]
*In XAML:*

[source,xaml]
----
<ig:{SparklineName} ToolTipVisibility="Visible" />
----
*In C#:*

[source,csharp]
----
this.{SparklineName}1.ToolTipVisibility = System.Windows.Visibility.Visible;
----

*In Visual Basic:*

[source,vb]
----
Me.{SparklineName}1.ToolTipVisibility = System.Windows.Visibility.Visible
----
endif::sl,wpf,win-phone,win-universal[] 
ifdef::win-forms[]
*In C#:*

[source,csharp]
----
this.{SparklineName}1.ToolTipVisibility = Infragistics.Portable.Components.UI.Visibility.Visible;
----

*In Visual Basic:*

[source,vb]
----
Me.{SparklineName}1.ToolTipVisibility = Infragistics.Portable.Components.UI.Visibility.Visible
----
endif::win-forms[]
