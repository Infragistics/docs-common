= API Overview ({ShapeChartName})

This section lists each namespace that is relevant to the link:{ShapeChartLink}.{ShapeChartName}.html[{ShapeChartName}] control. There are also several key classes listed that you will be using while programming with this control. The namespaces and classes on this page link directly into our API documentation.
