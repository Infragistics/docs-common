[[Introduction]]
== Introduction

The {FinancialChartName} provides a toolbar located at the top of the control.  This allows your users to change various details about the data being displayed in the chart.

pick:[jquery="image::images/financialchart_jquery_toolbar.png[]"]

[[DateRangeSelector]]
== Date Range Selector

The date range selector in the toolbar allows users to view chart data from several popular range presetspick:[wpf=" or to choose their own date range using the `From` and `To` date editors"].

ifdef::wpf[]

The `From` and `To` date values will automatically update as the user modifies the link:financial-chart-pane-zoom.html[Zoom Pane].

endif::wpf[]

[[IndicatorSelector]]
== Indicators Selector

The Indicators Selector allows the user to select multiple financial indicators, overlays, trendlines, and volume types for display in the {FinancialChartName}.

pick:[wpf="image::images/financialchart_wpf_toolbar_indicatortype.png[]"]
pick:[jquery="image::images/financialchart_jquery_toolbar_indicatortype.png[]"]

Financial Indicators are often used by traders to measure changes and to show trends in stock prices. These indicators are displayed below the link:financial-chart-pane-price.html[Price Pane] because they do not share the same Y-Axis scale.

ifdef::wpf[]

See {DataChartName}'s link:datachart-financial-indicators-overview.html[Financial Indicators] for more info on financial indicator types.
endif::wpf[]

Financial Overlays are often used by traders to measure changes and to show trends in stock prices. These overlays are displayed in front of the Financial Price Series because they share the same Y-Axis scale.

ifdef::wpf[]

See {DataChartName}'s link:datachart-financial-overlays-overview.html[Financial Overlays] for more info on financial overlay types.

endif::wpf[]

Trendlines are used to see a quick history of the trending path of your data.  The selected trendline is rendered on the link:financial-chart-layer-trendline.html[Trendline Layer].

The trendline offers several options to your users for displaying a trendline using various algorithms in the link:financial-chart-pane-price.html[Price Pane] and link:financial-chart-pane-volume.html[Volume Pane].

The volume type allows users to change how the volume of stock data is displayed in the link:financial-chart-pane-volume.html[Volume Pane].  Users can choose between None, Column, Line, and Area chart types.

[[ChartTypeSelector]]
== Chart Type Selector

The {FinancialChartName} can display data as four different chart types in the link:financial-chart-pane-price.html[Price Pane].  Users can choose between Bar, Line, Candle, and Column chart types.
