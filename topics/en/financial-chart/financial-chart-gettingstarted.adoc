[[Steps]]
=== Steps

The following procedure takes you through the steps necessary for adding the  {FinancialChartName} control to a page.

=== 1. Add the {FinancialChartName} control to the page.

The following code demonstrates adding the {FinancialChartName} control with the minimum code and property settings required for display.

ifdef::xaml[]
*In XAML:*
[source,xaml]
----
<Grid x:Name=”LayoutRoot”>
 <ig:{FinancialChartName} ItemsSource="{Binding DataCollection}" />
</Grid>
----
endif::xaml[]

ifdef::wpf[]
*In Visual Basic:*
[source,vb]
----
Dim financialChart As New {FinancialChartName}()
financialChart.ItemsSource = data
Me.LayoutRoot.Children.Add(financialChart)  
----

*In C#:*
[source,csharp]
----
var financialChart = new {FinancialChartName}();
financialChart.ItemsSource = data;
this.LayoutRoot.Children.Add(financialChart);  
----
endif::wpf[]

ifdef::jquery[]
Add an element for the {FinancialChartName} to be attached:


*In HTML*
[souce,html]
----
<body>
    <div id="financialChart"></div>
</body>
----

Attach the {FinancialChartName} to the element:


*In JavaScript*
[source,javascript]
----
$("#financialChart").{FinancialChartName}({ dataSource: data });
----
endif::jquery[]

=== 2. Verify the results.

Build and run your project to verify the result. If you have implemented the steps correctly, the displayed {FinancialChartName} should look like the one in the Preview section above.
