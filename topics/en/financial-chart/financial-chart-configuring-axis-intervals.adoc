= Axis Intervals

=== Purpose
This topic demonstrates how to configure the major and minor intervals for chart axes of the link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] control. 

In the {FinancialChartName} control, axis major interval specifies how frequent major gridlines and axis labels are rendered on an axis. Similarly, axis minor interval specifies how frequent minor gridlines are rendered on an axis.

== Configuring XAxis and YAxis Gridlines

[options="header", cols="a,a,a"]
|====
|In order to configure|Use this property|And set it to

|The frequency of major interval gridlines	
|link:{FinancialChartLink}.{FinancialChartName}{ApiProp}yAxisInterval{HtmlApiExtension}[pick:[wpf="YAxisInterval"]pick:[jquery="yAxisInterval"]]
|This value provides configurable spacing for y-axis labels and major gridlines. Note that the interval for axis labels will also be set by this value, displaying one label at the point on the axis associated with the interval.

On the {FinancialChartName} control's axes, this value is represented as a number between the axis minimum value and axis maximum value. By default, this axis will automatically calculate and find a nice and round interval based on axis minimum values and maximum value.

[[CodeSnippet]]
== Code Snippet

The following code snippet demonstrates how to configure the interval on the Y-axis.

ifdef::jquery[]
*In JavaScript:*
[source,javascript]
----
$(".selector").{FinancialChartName}({
    dataSource: data,
    yAxisInterval: 400,
    yAxisMajorStroke: "Green",
    yAxisMinorStroke: "Blue",
    yAxisMajorStrokeThickness: 3,
    yAxisMinorStrokeThickness: 1
});
----
endif::jquery[]

ifdef::wpf[]
*In XAML:*
[source,xaml]
----
<ig:{FinancialChartName} x:Name="chart1"
    YAxisInterval="400"
    YAxisMajorStroke="Green"
    YAxisMinorStroke="Blue"
    YAxisMajorStrokeThickness="3"
    YAxisMinorStrokeThickness="1" >
</ig:{FinancialChartName}>
----
endif::wpf[]

ifdef::wpf[]
*In C#:*
[source, csharp]
----
{FinancialChartName} chart1 = new {FinancialChartName}();
chart1.YAxisInterval = 400;
chart1.YAxisMajorStroke = Brushes.Green;
chart1.YAxisMinorStroke = Brushes.Blue;
chart1.YAxisMajorStrokeThickness = 3;
chart1.YAxisMinorStrokeThickness = 1;
----
endif::wpf[]

ifdef::wpf[]
*In Visual Basic:*
[source, vbnet]
----
Dim chart1 As {FinancialChartName} = new {FinancialChartName}()
chart1.YAxisInterval = 400
chart1.YAxisMajorStroke = Brushes.Green
chart1.YAxisMinorStroke = Brushes.Blue
chart1.YAxisMajorStrokeThickness = 3
chart1.YAxisMinorStrokeThickness = 1
----
endif::wpf[]
