= Configuring Axis Labels

The link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] control allows you full control over configuring, formatting and styling the labels displayed along the x-axis and y-axis on your chart. By default, you do not need to explicitly set the labels.

[[AbbreviatingLargeNumbers]]
=== Abbreviating Large Numbers

The link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] control allows you to specify if the y-axis labels should display abbreviated numeric values on the y-axis.  For label values that may appear too long, you can set link:{FinancialChartLink}.{FinancialChartName}{ApiProp}yAxisAbbreviateLargeNumbers{HtmlApiExtension}[YAxisAbbreviateLargeNumbers] to true to enable this feature.  This is set to *_true_* by default.

[[CodeSnippet]]
=== Code Snippet

The following code example shows how to style labels on the x-axis using date formatting and other style properties:

ifdef::jquery[]
*In JavaScript:*
[source,javascript]
----
$(".selector").{FinancialChartName}({
    dataSource: data,
    xAxisLabelLeftMargin: 14,
    xAxisLabelTextColor: "Green",
    yAxisLabelTextColor: "Blue"
});
----
endif::jquery[]

ifdef::wpf[]
*In XAML:*
[source,xaml]
----
<ig:{FinancialChartName} x:Name="chart1"  
    XAxisLabelFontSize="16"
    XAxisLabelForeground="Blue"
    XAxisLabelFontWeight="Bold"
    YAxisLabelFontSize="16"
    YAxisLabelForeground="Green"
    YAxisLabelFontWeight="Bold">
</ig:{FinancialChartName}>
----

endif::wpf[]

ifdef::wpf[]

*In C#:*
[source,csharp]
----
var chart1 = new {FinancialChartName}()
chart1.XAxisLabelFontSize = 16;
chart1.XAxisLabelForeground = new SolidColorBrush(Colors.Blue);
chart1.XAxisLabelFontWeight = FontWeights.Bold;
chart1.YAxisLabelFontSize = 16;
chart1.YAxisLabelForeground = new SolidColorBrush(Colors.Green);
chart1.YAxisLabelFontWeight = FontWeights.Bold;
----


*In Visual Basic:*
[source,vb]
----
Dim chart1 = New {FinancialChartName}()
chart1.XAxisLabelFontSize = 16
chart1.XAxisLabelForeground = New SolidColorBrush(Colors.Blue)
chart1.XAxisLabelFontWeight = FontWeights.Bold
chart1.YAxisLabelFontSize = 16
chart1.YAxisLabelForeground = New SolidColorBrush(Colors.Green)
chart1.YAxisLabelFontWeight = FontWeights.Bold
----
endif::wpf[]
