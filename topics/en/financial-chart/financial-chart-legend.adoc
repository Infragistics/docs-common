= Legend

=== Purpose

This topic demonstrates usage of the built-in Legend for the link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}].

[[Intro]]
=== Introduction

The link:{FinancialChartLink}.{FinancialChartName}.html[{FinancialChartName}] has a built-in link:{DataChartLink}.FinancialLegend.html[FinancialLegend] that once enabled, will appear for each series and overlay in the chart with its items horizontally aligned between the toolbar and the chart's plot area. The legend is disabled by default.

In order to enable the legend, you can set the link:{FinancialChartLink}.{FinancialChartName}{ApiProp}IsLegendVisible.html[IsLegendVisible] pick:[wpf="property"]pick:[jquery="option"] of the {FinancialChartName} to true.

[[CodeSnippet]]
=== Code Snippet

The following code can be used to enable the legend in the {FinancialChartName}.

ifdef::wpf[]

*In XAML:*
[source, xaml]
----
<ig:{FinancialChartName} x:Name="chart" ItemsSource="{Binding SelectedStock}" IsLegendVisible="True"/>
----

*In C#:*
[source, csharp]
----
chart.IsLegendVisible = true;
----

*In Visual Basic*
[source, vb]
----
chart.IsLegendVisible = True
----

endif::wpf[]

ifdef::jquery[]

*In JavaScript:*
[source, javascript]
----
$(function () {
    $("#chart").{FinancialChartName}({
        dataSource: stocks,
        isLegendVisible: true
    });
});
----

endif::jquery[]
