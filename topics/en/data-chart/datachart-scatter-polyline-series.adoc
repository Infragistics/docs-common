== Scatter Polyline Series

[[_Ref320193474]]

=== Overview

In the __{DataChartName}__ control, the link:{DataChartLink}.ScatterPolylineSeries_members.html[ScatterPolylineSeries] is a series that displays data using polylines. This type of scatter series is often used where rendering disconnected lines are required such as a network graph. The link:{DataChartLink}.ScatterPolylineSeries_members.html[ScatterPolylineSeries] works a lot like the link:{DataChartLink}.ScatterPolygonSeries_members.html[ScatterPolygonSeries] except that data is rendered with polylines instead of polygons.

=== Data Requirements

Similar to other types of series in the __{DataChartName}__ control, the link:{DataChartLink}.ScatterPolylineSeries_members.html[ScatterPolylineSeries] has the link:{DataChartLink}.series~{ApiDataSource}.html[{ApiDataSource}] property for the purpose of data binding. This property can be bound to an object that implements the link:http://msdn.microsoft.com/en-us/library/system.collections.ienumerable.aspx[IEnumerable] interface (e.g. link:http://msdn.microsoft.com/en-us/library/6sh2ey19.aspx[List], link:http://msdn.microsoft.com/en-us/library/ms132397.aspx[Collection], link:http://msdn.microsoft.com/en-us/library/7977ey2c.aspx[Queue], link:http://msdn.microsoft.com/en-us/library/system.collections.stack.aspx[Stack]). In addition, each item in this object must have one data column that stores point locations (X and Y values) of connected items using  link:http://msdn.microsoft.com/en-us/library/system.collections.ienumerable.aspx[IEnumerable]<link:http://msdn.microsoft.com/en-us/library/system.collections.ienumerable.aspx[IEnumerable]<pick:[wpf="link:http://msdn.microsoft.com/en-us/library/system.windows.point.aspx[Point]"]pick:[win-forms="{GeoMapNamespace}.Point"]>> structure. This data column is then mapped to the link:{DataChartLink}.ShapeSeriesBase~ShapeMemberPath.html[ShapeMemberPath] property. The link:{DataChartLink}.ScatterPolylineSeries_members.html[ScatterPolylineSeries] uses the points of this mapped data column to plot polyines in the control.

[[_Ref320651790]]

=== Example

In the above screenshot the structure of the data looks like this:

*In C#:*
[source,csharp]
----
public class Node
{
    // The lines that branch off this node to other nodes.
    public List<List<Point>> Points { get; set; }
}
----

*In Visual Basic:*
[source,vb]
----
Public Class Node
    ' The lines that branch off this node to other nodes.
    Private _points As List(Of List(Of Point))
    public Property List(Of List(Of Point)) Points
        Get
            Return _points
        End Get
        Set
            _points = value
        End Set
    End Property
End Class
----

The following code shows how to bind the link:{DataChartLink}.ScatterPolylineSeries_members.html[ScatterPolylineSeries] to data.

ifdef::wpf[]
*In XAML:*

[source,xaml]
----
<ig:{DataChartName} x:Name="dataChart">
    <ig:{DataChartName}.Axes>
        <ig:NumericXAxis x:Name="xAxis"/>
        <ig:NumericYAxis x:Name="yAxis"/>
    </ig:{DataChartName}.Axes>
    <ig:{DataChartName}.Series>
        <ig:ScatterPolylineSeries ItemsSource="{Binding Path=Nodes}"
                                  ShapeMemberPath="Points"
                                  XAxis="{Binding ElementName=xAxis}"
                                  YAxis="{Binding ElementName=yAxis}"/>
    </ig:{DataChartName}.Series>
</ig:{DataChartName}>
----
endif::wpf[]

*In Visual Basic:*

[source,vb]
----
' ScatterPolylineSeries requires numeric X and Y axis
Dim xAxis = New NumericXAxis()
Dim yAxis = New NumericYAxis()
Me.dataChart.Axes.Add(xAxis)
Me.dataChart.Axes.Add(yAxis)

' create and set data binding to the ScatterPolylineSeries
Dim polylineSeries = New ScatterPolylineSeries()
polylineSeries.{ApiDataSource} = Nodes
polylineSeries.ShapeMemberPath = "Points"
polylineSeries.XAxis = xAxis
polylineSeries.YAxis = yAxis

' add the ScatterPolylineSeries to the the {DataChartName}
Me.dataChart.Series.Add(polylineSeries)
----

*In C#:*

[source,csharp]
----
// ScatterPolylineSeries requires numeric X and Y axis
var xAxis = new NumericXAxis();
var yAxis = new NumericYAxis();
this.dataChart.Axes.Add(xAxis);
this.dataChart.Axes.Add(yAxis);

// create and set data binding to the ScatterPolylineSeries
var polylineSeries = new ScatterPolylineSeries();
polylineSeries.{ApiDataSource} = Nodes;
polylineSeries.ShapeMemberPath = "Points";
polylineSeries.XAxis = xAxis;
polylineSeries.YAxis = yAxis;

// add the ScatterPolylineSeries to the the {DataChartName}
this.dataChart.Series.Add(polyLineSeries);
----
