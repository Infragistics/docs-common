== Chart Overview Pane

=== Introduction

In the  _{DataChartName}_   control, an overview of chart content is represented by the {OpdName} control. This control provides a preview of all types of supported series as well as navigation functionalities.

=== Preview

The following is previews of the _{OpdName}_ control with an overview of the  _{DataChartName}_   control displaying financial data using link:{DataChartLink}.financialpriceseries_members.html[FinancialPriceSeries].

      OverviewPlusDetailPaneVerticalAlignment="Top">

== User Interaction and Usability

=== User interactions summary

The following table summarizes the user interaction capabilities of the {OpdName} control.

[options="header", cols="a,a,a"]
|====
|The user can…|Using…|Details

|Zoom a chart incrementally
|The mouse scroll wheel or zoom in/out buttons of the {OpdName} control.
|Clicking on the zoom in/out buttons of the {OpdName} control or scrolling mouse wheel over the preview area zooms in/out the chart content.

|Zoom a chart to a specific level
|The zoom slider of the {OpdName} control.
|Changing the zoom slider of the _{OpdName}_ control zooms in/out the chart content to a specific zoom level.

|Reset a chart to 100% zoom level
|The zoom reset button of the {OpdName} control.
|Clicking on the zoom reset button of the _{OpdName}_ control resets the chart to its 100% zoom level.

|Pan chart in all directions
|The window rectangle of the {OpdName} control.
|Dragging the window rectangle in preview area of the _{OpdName}_ control pans the chart content in a given direction.

|Pan to a specific region of a chart
|The preview area of the {OpdName} control.
|Clicking outside of the window rectangle in preview area of the _{OpdName}_ control pans the chart content to chart region where the mouse cursor was clicked.

|Change mouse drag interaction with a chart
|The cursor button of the {OpdName} control.
|Clicking on the cursor button of the _{OpdName}_ control changes mouse drag interaction with chart to pan on mouse dragging or to zoom on mouse dragging interactions.

|====
