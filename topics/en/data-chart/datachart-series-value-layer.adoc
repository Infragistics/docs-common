[[Overview]]
== Overview

=== Value Layer Overview

The link:{DataChartLink}.valuelayer.html[ValueLayer] is a series type that can be added to the {DataChartName}. The purpose of this series type is to annotate different focal points of your data such as the minimum, maximum, or average values by drawing a dashed line at those calculated points.

To determine which focal points you want to annotate, you can set the link:{DataChartLink}.valuelayer{ApiProp}ValueMode.html[ValueMode] property of the link:{DataChartLink}.valuelayer.html[ValueLayer]. This property takes one of the link:{DataChartLink}.valuelayer.html[ValueLayerValueMode] enumerations described below:

- `Auto`: The default value mode of the link:{DataChartLink}.valuelayer.html[ValueLayerValueMode] enumeration.
- `Average`: Applies potentially multiple value lines to call out the average value of each series plotted in the chart.
- `GlobalAverage`: Applies a single value line to call out the average of all of the series values in the chart.
- `GlobalMaximum`: Applies a single value line to call out the absolute maximum value of all of the series values in the chart.
- `GlobalMinimum`: Applies a single value line to call out the absolute minimum value of all of the series values in the chart.
- `Maximum`: Applies potentially multiple value lines to call out the maximum value of each series plotted in the chart.
- `Minimum`: Applies potentially multiple value lines to call out the minimum value of each series plotted in the chart.

You can configure the link:{DataChartLink}.valuelayer.html[ValueLayer] to target a specific series if you want to have multiple layers present with different configurations. This can be done be setting the link:{DataChartLink}.valuelayer{ApiProp}TargetSeries.html[TargetSeries] property on the layer.

[[Properties]]
== Properties

=== Value Layer Summary

The following table summarizes the properties of the link:{DataChartLink}.valuelayer.html[ValueLayer] layer.

[options="header", cols="a,a,a"]
|====
|Property Name|Property Type|Description

| link:{DataChartLink}.valuelayer{ApiProp}IsAxisAnnotationEnabled.html[IsAxisAnnotationEnabled]
|`bool`
|Determines whether or not an annotation on the axis should be visible for each of the value layer lines. 

| link:{DataChartLink}.valuelayer{ApiProp}TargetSeries.html[TargetSeries]
| `Series`
| The series that the link:{DataChartLink}.valuelayer.html[ValueLayer] lines should take into account when determining the placement of the lines for the link:{DataChartLink}.valuelayer{ApiProp}ValueMode.html[ValueMode].

| link:{DataChartLink}.valuelayer{ApiProp}ValueMode.html[ValueMode]
| link:{DataChartLink}.valuelayer.html[ValueLayerValueMode]
| The type of link:{DataChartLink}.valuelayer.html[ValueLayer] lines that should be applied.

|====

[[Example]]
== Example

Following is the code used in the screenshot above.

ifdef::wpf[]

*In XAML:*

[source,xaml]
----
<ig:{DataChartName} x:Name="chart">
    <ig:{DataChartName}.Axes>
        <ig:CategoryXAxis x:Name="xAxis" ItemsSource="{Binding Data}" Label="{}{Label}" />
        <ig:NumericYAxis x:Name="yAxis" />
    </ig:{DataChartName}.Axes>
    <ig:{DataChartName}.Series>
        <ig:ColumnSeries XAxis="{Binding ElementName=xAxis}"
                         YAxis="{Binding ElementName=yAxis}"
                         ItemsSource="{Binding Data}"
                         ValueMemberPath="temperatureHigh" />
                         
        <ig:ColumnSeries XAxis="{Binding ElementName=xAxis}"
                         YAxis="{Binding ElementName=yAxis}"
                         ItemsSource="{Binding Data}"
                         ValueMemberPath="temperatureLow" />

        <ig:ValueLayer ValueMode="Maximum" IsAxisAnnotationEnabled="True"/>

    </ig:{DataChartName}.Series>
</ig:{DataChartName}>
----

endif::wpf[]

ifdef::wpf[]
*In C#:*
[source,cs]
----
var layer = new ValueLayer();
layer.IsAxisAnnotationEnabled = true;
layer.ValueMode = ValueLayerValueMode.Maximum;
DataChart.Series.Add(layer);
----
endif::wpf[]
