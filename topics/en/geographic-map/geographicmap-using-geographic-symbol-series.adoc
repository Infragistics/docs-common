== Geographic Symbol Series

[[_Ref320193474]]

=== Overview

In the _{GeoMapName}_ control, the link:{GeoMapLink}.geographicsymbolseries_members.html[GeographicSymbolSeries] is a visual map element that displays geo-spatial data using points or markers in a geographic context. This type of geographic series is often used to render a collection of geographic locations such as cities, airports, earthquakes, or points of interests.

[[_Ref320651783]]

=== Data Requirements

Similarly to other types of geographic series in the _{GeoMapName}_ control, the link:{GeoMapLink}.geographicsymbolseries_members.html[GeographicSymbolSeries] has the link:{DataChartLink}.series~{ApiDataSource}.html[{ApiDataSource}] property for the purpose of data binding. This property can be bound to an object that implements the link:http://msdn.microsoft.com/en-us/library/system.collections.ienumerable.aspx[IEnumerable] interface (e.g. link:http://msdn.microsoft.com/en-us/library/6sh2ey19.aspx[List], link:http://msdn.microsoft.com/en-us/library/ms132397.aspx[Collection], link:http://msdn.microsoft.com/en-us/library/7977ey2c.aspx[Queue], link:http://msdn.microsoft.com/en-us/library/system.collections.stack.aspx[Stack]). In addition, each item in this object must have two numeric data columns that store a geographic location (longitude and latitude). These data columns are then mapped to the link:{GeoMapLink}.geographicsymbolseries~latitudememberpath.html[LatitudeMemberPath] and link:{GeoMapLink}.geographicsymbolseries~longitudememberpath.html[LongitudeMemberPath] properties. The link:{GeoMapLink}.geographicsymbolseries_members.html[GeographicSymbolSeries] uses values of these mapped data columns to plot symbol elements in the _{GeoMapName}_ control.

[[_Ref320651790]]

=== Example

The following code shows how to bind the link:{GeoMapLink}.geographicsymbolseries_members.html[GeographicSymbolSeries] to locations of cities loaded from a shape file using the ShapefileConverter.

ifdef::wpf[]
*In XAML:*

[source,xaml]
----
<ig:ShapefileConverter x:Key="shapefileConverter"
                        ShapefileSource="ShapeFiles/world_cities.shp"
                        DatabaseSource="ShapeFiles/world_cities.dbf" >
</ig:ShapefileConverter>
<ig:XamGeographicMap x:Name="GeoMap">
    <ig:XamGeographicMap.Series>
        <ig:GeographicSymbolSeries {ApiDataSource}="{StaticResource shapefileConverter}"
                                   LongitudeMemberPath="Points[0][0].X"
                                   LatitudeMemberPath="Points[0][0].Y">
        </ig:GeographicSymbolSeries>
    </ig:XamGeographicMap.Series>
</ig:XamGeographicMap>
----
endif::wpf[]

*In Visual Basic:*

[source,vb]
----
' create and set data binding to the GeographicSymbolSeries
Dim geoSeries = New GeographicSymbolSeries()
geoSeries.{ApiDataSource} = shapefileConverter
geoSeries.LongitudeMemberPath = "Points[0][0].X"
geoSeries.LatitudeMemberPath = "Points[0][0].Y"
' add the GeographicSymbolSeries to the the {GeoMapName}
Me.GeoMap.Series.Add(geoSeries)
----

*In C#:*

[source,csharp]
----
// create and set data binding to the GeographicSymbolSeries
var geoSeries = new GeographicSymbolSeries();
geoSeries.{ApiDataSource} = shapefileConverter;
geoSeries.LongitudeMemberPath = "Points[0][0].X";
geoSeries.LatitudeMemberPath = "Points[0][0].Y";
// add the GeographicSymbolSeries to the the {GeoMapName}
this.GeoMap.Series.Add(geoSeries);
----
