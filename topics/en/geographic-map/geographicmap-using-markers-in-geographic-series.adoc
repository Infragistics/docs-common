== Displaying Basic Markers in Geographic Series

[[_Ref320745641]]

=== Overview

In the _{GeoMapName}_ control, markers are visual elements that display values of data items bound to geographic series in geographic locations of the map. Marker can be represented as labels, symbols or custom data templates.

The following types of geographic series support markers:

* link:{GeoMapLink}.geographicsymbolseries_members.html[GeographicSymbolSeries]
* link:{GeoMapLink}.geographicshapeseries_members.html[GeographicShapeSeries]

[[_Ref320745644]]

=== Code

The following code changes the appearance of markers in the link:{GeoMapLink}.geographicsymbolseries_members.html[GeographicSymbolSeries]

ifdef::wpf[]
*In XAML:*

[source,xaml]
----
<ig:XamGeographicMap.Series>
    <ig:GeographicSymbolSeries ItemsSource="{StaticResource shapefileConverter}"
                                MarkerBrush="Blue" 
                                MarkerOutline="Black"
                                MarkerCollisionAvoidance="None"
                                LongitudeMemberPath="Points[0][0].X"
                                LatitudeMemberPath="Points[0][0].Y">
    </ig:GeographicSymbolSeries>
</ig:XamGeographicMap.Series>
----
endif::wpf[]

*In C#:*
[source,c#]
----
var geoSeries = new GeographicSymbolSeries();
geoSeries.{ApiDataSource} = shapefileConverter;
geoSeries.MarkerBrush = new SolidBrush(Color.Blue);
geoSeries.MarkerOutline = new SolidBrush(Color.Black);
geoSeries.MarkerCollisionAvoidance = CollisionAvoidanceType.None;
geoSeries.LatitudeMemberPath = "Points[0][0].X";
geoSeries.LongitudeMemberPath = "Points[0][0].Y";
this.geoMap.Series.Add(geoSeries);
----

*In VB:*
[source,c#]
----
Dim geoSeries = new GeographicSymbolSeries()
geoSeries.{ApiDataSource} = shapefileConverter
endif::win-forms[]
geoSeries.MarkerBrush = new SolidBrush(Color.Blue)
geoSeries.MarkerOutline = new SolidBrush(Color.Black)
geoSeries.MarkerCollisionAvoidance = CollisionAvoidanceType.None
geoSeries.LatitudeMemberPath = "Points[0][0].X"
geoSeries.LongitudeMemberPath = "Points[0][0].Y"
Me.geoMap.Series.Add(geoSeries)
----

== Creating Custom Markers in Geographic Series

[[_Ref320745659]]

=== Overview

The link:{GeoMapLink}.geographicshapeseries~markertemplate.html[MarkerTemplate] property of geographic series can be used to provide custom data templates to display custom symbols and/or values of data items. Custom data templates can access values of data items bound to the {ApiDataSource} property of geographic series by binding to the Item property. When a geographic series is bound to the link:{DataVizMapsLink}.shapefileconverter_members.html[ShapefileConverter] then the Item property is resolved to a link:{DataVizMapsLink}.shapefilerecord_members.html[ShapefileRecord] object and values loaded from a shape database (.dbf) file can be accessed through the link:{DataVizMapsLink}.shapefilerecord~fields.html[Fields] property.

[[_Ref320745688]]

=== Preview

The following is a preview of the _{GeoMapName}_ control with markers displaying names of countries on all shapes of link:{GeoMapLink}.geographicshapeseries_members.html[GeographicShapeSeries].

image::images/GeographicMap_Using_Markers_in_Geographic_Series_2.png[]

=== Example

The following code creates a custom data template for markers of link:{GeoMapLink}.geographicshapeseries_members.html[GeographicShapeSeries] with binding to a data column that contains names of countries.

*In XAML:*

[source,xaml]
----
<ig:GeographicShapeSeries {ApiDataSource}="{StaticResource shapefileConverter}"
                          ShapeMemberPath="Points"
                          MarkerCollisionAvoidance="Fade">
    <ig:GeographicShapeSeries.MarkerTemplate>
        <DataTemplate>
            <TextBlock Text="{Binding Path=Item.Fields[CNTRY_NAME]}" FontWeight="Bold" />
        </DataTemplate>
    </ig:GeographicShapeSeries.MarkerTemplate>
</ig:GeographicShapeSeries>
----

endif::wpf[]
