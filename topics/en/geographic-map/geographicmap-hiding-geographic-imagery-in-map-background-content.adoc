== Map Background Content

[[_Ref320193474]]

=== Overview

By the default, the _{GeoMapName}_ control displays geographic imagery from the Open Street Maps in the map background content using the link:{GeoMapLink}.openstreetmapimagery_members.html[OpenStreetMapImagery] class. However, there might be scenarios (for example, geographic shape series with detailed shape files of the world) where geo-spatial data from shape files provides enough geographic contexts and geographic imagery is not desired in the map background content.

[[_Ref320388681]]

=== Example

This code example hides geographic imagery by setting the _{GeoMapName}_ control’s pick:[wpf="link:{GeoMapLink}.xamgeographicmap~backgroundcontent.html[BackgroundContent]"]pick:[win-forms="BackgroundContent"] to a null value.

ifdef::wpf[]
*In XAML:*

[source,xaml]
----
<ig:XamGeographicMap x:Name="GeoMap" BackgroundContent="{x:Null}">
    <ig:XamGeographicMap.Series>
        <!-- add geographic series here -->
    </ig:GeographicShapeSeries>
</ig:XamGeographicMap>
----
endif::wpf[]

*In Visual Basic:*

[source,vb]
----
ifdef::wpf[]
Dim geoMap As New XamGeographicMap()
Me.geoMap.BackgroundContent = Nothing
endif::wpf[]
ifdef::win-forms[]
Dim geoMap As New UltraGeographicMap()
Me.geoMap.ResetBackgroundContent()
Me.geoMap.Refresh()
endif::win-forms[]
----

*In C#:*

[source,csharp]
----
ifdef::wpf[]
XamGeographicMap geoMap = new XamGeographicMap();
this.geoMap.BackgroundContent = null;
endif::wpf[]
ifdef::win-forms[]
UltraGeographicMap geoMap = new UltraGeographicMap();
this.geoMap.ResetBackgroundContent();
this.geoMap.Refresh();
endif::win-forms[]
----
