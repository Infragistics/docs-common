=== Visual elements and related properties

The following table maps the visual elements of the _{GeoMapName}_ control and the properties that configure them.

[options="header", cols="a,a,a"]
|====
|Visual Element|Description|Property

|Map Plot Area
|Represents an area within the _{GeoMapName}_ control used for plotting geographic imagery tiles and geo-spatial data using geographic series as well as displaying map crosshair lines pick:[wpf="and map overview pane"]. Background of this area can changed to any pick:[wpf="link:http://msdn.microsoft.com/en-us/library/system.windows.media.brush.aspx[Brush]"]pick:[win-forms="Infragistics.Win.DataVisualization.Brush"] object by setting the link:{DataChartLinkBase}.seriesviewer~plotareabackground.html[PlotAreaBackground] property of the _{GeoMapName}_ control.
| link:{DataChartLinkBase}.seriesviewer~plotareabackground.html[PlotAreaBackground]

|Map Background Content
|Represents an area of the _{GeoMapName}_ control for plotting geographic imagery tiles from supported mapping services such as Open Street Mappick:[wpf", ArcGIS Maps,"] and Azure Maps. Refer to the link:geographicmap-using-geographic-imagery.html[Using Geographic Imagery] topic for more information about this map element.
| pick:[wpf="link:{GeoMapLink}.{GeoMapName}{ApiProp}backgroundcontent.html[BackgroundContent]"]pick:[win-forms="link:{GeoMapLink}.{GeoMapName}{ApiProp}SetImagery.html[SetImagery()]"]

|Geographic Series
|Represent map elements for rendering geo-spatial vector data -points ( link:{GeoMapLink}.geographicsymbolseries_members.html[GeographicSymbolSeries]), polylines ( link:{GeoMapLink}.geographicpolylineseries_members.html[GeographicPolylineSeries]), or polygons ( link:{GeoMapLink}.geographicshapeseries_members.html[GeographicShapeSeries]) in geographic context. The _{GeoMapName}_ control’ link:{DataChartLinkBase}.seriesviewer~series.html[Series] property is a collection of geographic series objects. For more information about geographic series elements supported in the _{GeoMapName}_ control, refer to the link:geographicmap-using-geographic-series.html[Using Geographic Series] topic.
| link:{DataChartLinkBase}.seriesviewer~series.html[Series]

|Map Crosshair Lines
|Represents vertical and horizontal lines that intersect at the location of mouse cursor when it hovers over the map plot area of the _{GeoMapName}_ control.
| link:{DataChartLinkBase}.seriesviewer~crosshairvisibility.html[CrosshairVisibility] pick:[wpf="link:{DataChartLinkBase}.seriesviewer~crosshairlinestyle.html[CrosshairLineStyle]"]

ifdef::wpf[]
|Map Overview Pane
|Represents the link:{ApiPlatform}{OpdAssembly}{ApiVersion}~{OpdNamespace}.{OpdName}_members.html[{OpdName}] control with configurable location and easy-to-use zooming and panning functionalities for navigating map content of the _{GeoMapName}_ control.
| link:{DataChartLinkBase}.seriesviewer~overviewplusdetailpanevisibility.html[OverviewPlusDetailPaneVisibility] link:{DataChartLinkBase}.seriesviewer~overviewplusdetailpanestyle.html[OverviewPlusDetailPaneStyle]
endif::wpf[]
